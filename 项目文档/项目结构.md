# GuandanKitty 项目结构

## 项目两大区域

### 🎨 设计区 — `项目文档/`
所有设计讨论、AI 协作产物均放此处。`项目文档/` 之外为纯 Godot 工程。

```
项目文档/
├── 项目结构.md              ← 本文件
├── 游戏设计/                 ← 游戏设计知识体系 & 工作流
│   ├── 能力清单.md           ← 共享资源总览（Agents/Skills/Prompts/Workflows/Scripts）
│   ├── agents/               ← 可调用的 AI 设计角色
│   │   └── 游戏设计师.agent.md
│   ├── prompts/              ← 面向 AI 的详细操作指令
│   │   ├── 卡牌设计.prompt.md
│   │   ├── 事件设计.prompt.md
│   │   ├── 武器设计.prompt.md
│   │   ├── 数值平衡.prompt.md
│   │   ├── 经验提炼.prompt.md
│   │   ├── 设计工作流.prompt.md
│   │   ├── 设计自检.prompt.md
│   │   └── 需求理解与上下文采集.prompt.md
│   ├── 技能/                 ← 领域知识 & 方法论沉淀（.skill.md）
│   │   ├── canvas生成.skill.md
│   │   ├── 专业/             ← 专业领域技能（框架设计、框架转GDD等）
│   │   ├── 工具/

│   ├── 工作流/               ← 可执行工作流定义（JSON）+ 引擎（Python）
│   │   ├── 设计引擎.py       ← 工作流执行引擎
│   │   ├── 设计路由.json     ← 工作流路由定义
│   │   ├── 设计状态.json     ← 当前工作流状态
│   │   └── *.json            ← 各工作流定义（GDD分析、框架设计、知识入库等）
│   ├── 知识库/               ← 游戏设计理论知识

│   │   ├── UI设计理论/       ← 游戏 UI/UX 专题
│   │   ├── 项目经验/         ← 单项目经验
│   │   ├── 跨项目知识/       ← 多项目验证的通用规则
│   │   └── 已验证经验/       ← 审核通过的可靠经验
│   ├── 脚本/                 ← 自动化运维脚本
│   ├── GUANDANKITTY/         ← 本项目专属设计文档
│   │   └── 核心玩法文档
│   └── _验证报告/            ← 验证报告归档
└── 程序相关/                 ← （待建设）程序相关协作文档
```

### 🛠️ 工程区 — Godot 4.6.2 C# 工程
```
GuandanKitty.sln          # 解决方案
GuandanKitty.csproj       # 项目文件 (Godot.NET.Sdk/4.6.2, net8.0)
project.godot             # Godot 工程配置

Core/                     # 核心逻辑层
├── BattleManager.cs      # 战斗管理器（回合制流程）
├── Card.cs               # 卡牌基类 (Resource)
├── CardDatabase.cs       # 卡牌数据库
├── Character.cs          # 角色基类
├── Enemy.cs              # 敌人逻辑
├── MapGenerator.cs       # 地图生成
├── Player.cs             # 玩家（手牌、能量、抽牌堆等）
└── StatusEffect.cs       # 状态效果

UI/                       # UI 层
├── BattleUI.cs/tscn      # 战斗界面
├── CardUI.cs/tscn        # 卡牌预制体
├── EnemyIntentUI.cs/tscn # 敌人意图显示
├── HealthBar.cs/tscn     # 血条
├── MainMenuUI.cs/tscn    # 主菜单
├── MapNodeUI.cs/tscn     # 地图节点
└── MapUI.cs/tscn         # 地图界面

Scenes/                   # 场景文件
├── MainScene.cs/tscn     # 场景管理器（UI切换中枢）
├── TestScene.tscn        # 测试场景
├── CardDatabase.tscn     # 卡牌数据库场景
└── Enemy.tscn            # 敌人预制体

Docs/
└── UI_Specification.md   # UI 场景规范文档
```

## 关键技术信息
- **引擎**: Godot 4.6.2
- **语言**: C# (.NET 8.0, Android 用 net9.0)
- **核心玩法**: 类斗地主纸牌对战 Roguelike
- **UI 切换流**: MainMenuUI → MapUI → BattleUI → MapUI（循环）

## 工作约定
- 所有设计讨论、AI 协作产物均放入 `项目文档/`
- `项目文档/` 之外的部分为纯 Godot 工程文件，不混入协作内容
