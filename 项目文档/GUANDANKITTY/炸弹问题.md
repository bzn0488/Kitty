# 炸弹问题

> 状态：**搁置**（待玩法模型完善后重新评估） | 创建：2026-06-24
> 关联：`核心玩法文档` §2.3 出牌阶段
> 决策：优先寻找更优方案；若最终无更好方案，**回退到方案 A（炸弹不入牌河）**

---

## 一、问题定义

在 GuandanKitty 的基础规则中，炸弹（≥4 张同数字）是唯一能**跨牌型压制**的牌型——其他牌型必须同型对同型，但炸弹可以压制任何牌型。这让炸弹成为"终结技"：只要手中有炸弹且敌人没有更大的炸弹，玩家就能**确保赢得当前回合**。

### 炸弹与设计目标的冲突

按设计分析框架逐层检视：

### 1.1 正反馈层面

炸弹本身没有问题——打出炸弹是强烈的正反馈：

> 玩家攒出炸弹 → 打出 → 巨额伤害 → 满足感 ✓

问题不在炸弹的正反馈本身，而在于它**扭曲了其他行为的正反馈**：如果"诱敌接龙→炸弹收尾"总是最优，那么玩家在接龙过程中打出的每一张牌都不是"我在博弈"，而是"我在等炸弹"。博弈的正反馈被稀释。

### 1.2 进程层面

炸弹速胜推进进程很快（1 轮结束），不违反快乐密度原则。但炸弹+接龙组合（诱敌→炸弹收尾）让长接龙变成了**无风险**的高收益行为——这反而会让玩家觉得接龙过程是"沉闷的等待"而非"紧张的博弈"，进程中的快乐被稀释。

### 1.3 相对最优解层面（核心破坏）

炸弹对相对最优解空间的破坏是最严重的：

```
通解路径：
  随便出小牌 → 敌人跟 → 再出 → … → 炸弹收尾
  
  效果：
  · 牌河累积（所有牌面值）
  · 长接龙倍率（最高可达 5.5×）
  · 炸弹倍率（3.0×）
  · 零风险（炸弹必定碾压非炸弹，敌人手牌可见）
  → 最大收益 + 零风险 = 唯一最优解
```

**后果**：
- 所有策略收敛到"凑炸弹+拖接龙"——L2 解答方式变化坍缩为 1
- 敌人手牌可见的设计（本来是玩家博弈的工具）变成了"确认敌人没炸弹就不怕"的检查清单
- 叫牌只为了找炸弹，贴纸/遗物只需强化炸弹——构筑多样性被压扁
- 不同敌人的战败效果差异变得不重要（反正你不会输回合）

---

## 二、核心约束

任何解决方案必须满足：

1. **炸弹仍是正反馈**：打出炸弹必须有满足感，不能变成"诅咒"
2. **不破坏进程**：炸弹不能拖慢战斗节奏
3. **保留炸弹的保底价值**：炸弹应该是"可靠的选择"，但不应是"唯一最优的选择"
4. **不引入复杂规则**：裸规则层面保持直觉可理解

## 三、备选方案 A：炸弹不入牌河

### 机制

炸弹打出的牌**不进入牌河**，独立计算伤害：

```
炸弹伤害 = 炸弹牌面之和 × 炸弹倍率（3.0）
普通伤害 = 牌河总值 × 牌型倍率 × 接龙倍率

当炸弹作为终结技时：
  总伤害 = 牌河中非炸弹牌的伤害 + 炸弹独立伤害
```

### 设计意图

炸弹从"牌河的放大器"变为"独立的伤害来源"。如果玩家选择炸弹终结：

- 牌河中的牌仍然按普通规则计分（但失去高倍率牌型的加成）
- 炸弹本身稳定产出伤害，但不享受接龙倍率

### 数值对比

```
策略A（炸弹速胜）：
  出 8-8-8-8 → 1轮 → 炸弹终结
  伤害 = 32 × 3.0 = 96

策略B（诱敌5轮，顺子收尾）：
  牌河总值 ≈ 85
  伤害 = 85 × 2.0 × 4.0 = 680

策略C（诱敌5轮，炸弹收尾）：
  牌河(非炸弹) ≈ 42，炸弹 = 32
  伤害 = 42 × 1.0 × 4.0 + 32 × 3.0 = 168 + 96 = 264
```

策略C 的伤害（264）远低于策略B（680），玩家有动力挑战"不用炸弹收尾"。但策略C > 策略A（264 > 96），所以拖接龙+炸弹仍有收益，只是不再是通解。

### 优点

- 直观：炸弹是"砸场子"，砸场子不看前面铺垫了多少
- 保留炸弹的正反馈和保底价值
- 用数学差异而非额外规则来解决通解问题

### 缺点

- 炸弹的"不进牌河"是一个特殊规则，增加了认知负担
- 可能与未来贴纸设计冲突（例如"牌河中每张牌 +1 伤害"——炸弹不参与会让该贴纸对炸弹策略无效）
- 没有充分利用牌河的"赢家通吃"维度来制衡炸弹

## 四、备选方案 B：炸弹余威

### 机制

炸弹不再是最强终结技，而是**提供本场战斗永久增益**。越早打出炸弹，增益覆盖的回合越多，总收益越大。

```
炸弹伤害 = 炸弹牌面之和 × 炸弹倍率（3.0）

额外效果：打出炸弹后，本场战斗后续所有牌型获得
  「牌型倍率 +0.5」（多炸弹可叠加）
```

### 设计意图

把炸弹从"留着等收尾→通解"扭转为"早出早享受→加速器"：

```
旧模式：拖接龙 → 等炸弹收尾 → 同时享受接龙倍率+炸弹倍率
新模式：炸弹早出 → 永久增益 → 后续每轮都在滚雪球
```

### 数值对比

```
策略A（炸弹第1轮早出）：
  回合1：炸弹 8-8-8-8→32×3.0=96，余威+0.5 启动
  回合2：顺子 7-8-9-10-J(35)→35×2.5(顺子+余威)=87.5
  回合3：一对 Q-Q(24)→24×1.8(一对+余威)=43.2
  总伤 ≈ 226+ ——回合越多越赚

策略B（炸弹第3轮收尾，无余威利用）：
  回合1：单张 3→3×1.0=3
  回合2：一对 10-10→20×1.3=26
  回合3：炸弹 8-8-8-8→32×3.0=96
  总伤 = 125 ——显著低于策略A
```

玩家自己会算账：早出炸弹虽然这轮"浪费"了接龙倍率，但后面每轮都在倍数叠加，总体更赚。

### 优点

- **纯正反馈**：炸弹是纯粹的增益，不存在任何负面
- **不特殊规则**：炸弹仍入牌河，仍享赢家通吃，只是多了一个被动效果
- **开放构筑空间**：遗物"余威 +1.0"、贴纸"这张牌也触发余威"等设计方向自然涌现
- **叙事自洽**："炸弹震撼战场，余威持续整场战斗"

### 缺点

- 余威增益的效果可能在前期不明显（需要几轮才能体现差异），新手可能感觉不到
- 如果战斗中炸弹出现很晚（抽不到），余威无机会生效——需要配套的炸弹检索机制
- 多炸弹叠加上限需设定（防止余威飙到离谱）

## 五、待探索方向

以下方向尚未形成完整方案，但值得进一步思考：

- **炸弹消耗机制**：打出炸弹后，炸弹牌移出本场战斗（类似战败弃牌），让炸弹成为"一次性核武器"
- **牌河稀释**：炸弹进入牌河但降低接龙倍率（"炸弹打断了接龙的流畅性"——叙事上合理）
- **接龙倍率仅作用于同牌型**：如果接龙过程中牌型变化，倍率重置——让"诱敌→切炸弹"失去接龙倍率加成
