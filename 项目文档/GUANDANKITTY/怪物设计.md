## 一、核心概念：怪物 = 敌方的「遗物」

与杀戮尖塔（多独立敌人）不同，GuandanKitty 中**敌方是一个统一实体**——一个手牌区、一条血条。怪物是挂载在敌方身上的被动修正器，类似于玩家的遗物。

```
玩家侧                          敌方侧
┌──────────────┐              ┌──────────────┐
│  🃏 一副牌组   │              │  🃏 一副牌组   │
│  ✋ 一个手牌区 │              │  ✋ 一个手牌区 │
│  ❤️ 一条命    │              │  ❤️ 一条血    │
│              │              │              │
│  遗物（被动）  │              │  怪物（被动）  │
│  ├─ 遗物A    │              │  ├─ 怪物A    │
│  ├─ 遗物B    │              │  ├─ 怪物B    │
│  └─ 遗物C    │              │  └─ 怪物C    │
└──────────────┘              └──────────────┘
```

## 二、怪物属性字段（草案）

每个怪物作为敌方的被动挂载，提供以下属性：

| 字段 | 说明 | 示例 |
|------|------|------|
| **HP 贡献** | 叠加到敌方总血量 | 哥布林 +500 HP |
| **战败效果** | 敌方赢得回合时触发 | 哥布林：随机弃玩家 3 张手牌，补抽 3 张 |
| **初始手牌贡献** | 战斗开始时为敌方增加的手牌 | 哥布林：+3 张随机牌 |
| **每回合抽牌贡献** | 敌方回合结束时额外抽牌数 | 巨魔：每回合 +1 抽 |
| **特殊规则** | 被动改变敌方的行为或规则 | （待设计） |

> **HP 贡献**和**战败效果**是两个必填字段，其余可选。

## 三、开火车（难度自主调节）

### 3.1 规则

玩家在进入房间前可以选择**开火车**——跳过当前房间，将怪物带入下一场战斗。

```
地图：  🏠房间1    🏠房间2    🏠房间3
        哥布林     巨魔       哥布林×2
        500HP     800HP      1000HP

保守打法：逐个进入，3 场独立战斗
激进打法：房间1开火车 → 一波清 2300HP，3倍奖励
```

### 3.2 设计价值

| 问题 | 开火车如何解决 |
|------|------|
| 伤害数值狂暴（指数增长 + 连乘） | HP 由玩家自主调节——觉得伤害溢出就多拉怪 |
| 体验不可控感 | 控制权交还玩家——拉多少怪 = 选什么难度 |
| 风险收益对等 | 拉 N 个怪 = N 倍风险 + N 倍奖励 |
| 构筑强度表达 | 高手拉 5 怪一波清 = 构筑实力的直接证明 |

### 3.3 核心隐喻

取自 MMO 中的「开火车」——坦克一路拉怪聚一波 AOE 清掉。玩家在这里既是坦克（承担风险）又是 DPS（输出伤害），开火车的技术本身就是玩家技能的组成部分。

## 四、待讨论问题

### Q1：开火车上限

- ✅ **已确认**：最多拉 **5 波**怪物（防止数值完全失控）
- 5 波上限配合伤害公式的指数增长，可承受范围已被大幅扩展

### Q1.5：单房间难度差分

- ✅ **已确认**：每个怪物房提供**高难度 / 低难度**两个备选怪物
- 玩家在进入房间前选择难度，再决定是否从此房间开火车
- 二维难度控制：

```
横轴（开火车）：拉 N 个房间（1~5波）
纵轴（每房间）：选高难/低难怪物

组合空间：2⁵ = 32 种难度配置
```

- 即使不跨房间（不开火车），单房间内也有难度选择——不想拉怪的玩家仍有决策空间

### Q2：开火车后敌方手牌 & Q6：怪物手牌来源

- ✅ **已确认**：每个怪物有自己的**牌池**（~5 张），牌面构成定义其「牌风」，牌池中可以有重复牌面
- 每回合结束时，敌方从**每个怪物牌池中各抽 1 张复制品**（牌池不消耗——如同投骰子）
- 多个怪物 → 每回合抽多张 → 敌方手牌成长更快 → 压制力更强
- 强力怪物可能**每回合抽多张**（如龙抽 2 张）

| 怪物 | 牌池示例 | 抽牌速率 | 牌风 |
|------|------|:--:|------|
| 哥布林 | [♠2, ♥3, ♦3, ♣4, ♠5] | 1/回合 | 小杂牌，压制弱，接龙友好 |
| 巨魔 | [♠8, ♥9, ♦K, ♣K, ♠A] | 1/回合 | 大牌倾向，压制强，奖金池厚 |
| 龙 | [♥A, ♠A, ♦A, ♣A] | 2/回合 | 炸弹潜力，双倍成长，终极威胁 |

> 设计要点：
> - 牌池不消耗（抽复制品）→ 敌方手牌随回合线性/加速增长 → **逼玩家快攻，杜绝龟缩**
> - 牌池与玩家牌堆独立 → 双方的牌不互斥，叫牌机制不受怪物影响
> - 每回合抽牌在**回合结束阶段**执行（与现有规则一致），玩家出牌前能看到敌方全部手牌

### Q3：开火车时机

- ✅ **已确认**：仅在**进入战斗前**决定（风险与收益模型——战前 commit，战中承担）
- 不能在战斗中追加怪物

### Q4：奖励公式

- N 个怪物 = N 倍奖励（线性）？
- 还是 N 个怪物 = N× 基础 × 额外倍率（鼓励多拉）？
- 有没有特殊奖励（如一波清 3+ 怪额外掉稀有遗物）？

### Q5：战败效果叠加

- ✅ **已确认**：敌方有多个怪物时，赢回合后**全部战败效果同时触发**
- 零额外规则——叠加就是叠加
- 这是开火车风险超线性的核心来源之一（输一回合 = N 个效果砸脸）

### Q7：怪物与关卡层级

- ✅ **已确认**：怪物有等级，1 级有基础血量，每升 1 级 +10%（复利）
- 公式：`HP = 基础HP × 1.1^(等级−1)`

| 哥布林 (基础 500) | HP |
|:--:|:--:|
| Lv1 | 500 |
| Lv3 | 605 |
| Lv5 | 732 |
| Lv10 | 1,179 |

> 指数 HP 增长 + 伤害公式指数增长 → 两条曲线同频，难度不会随等级失控。
