对游戏中的数值体系进行平衡性分析。**基于理论计算和静态分析，不运行代码。**

## 使用时机

这是一个独立工具，**不作为工作流的固定步骤**。建议在以下场景使用：
- 方案设计完成后，想确认数值是否合理
- 设计过程中遇到数值决策，需要参考基线
- 新增职业/卡牌/事件后，检查与已有体系的协调性

## 分析维度

### 1. 经济循环分析
通过静态计算表检查：

```
基准产出计算：
  在职人数 × (基准产出 + 加成) × 税率 = 理论年收入
  被抚养人数 × 人均消耗 = 年支出
  净收入 = 年收入 - 年支出

示例（三国，10人，50%税率）：
  8人产出：8 × (1.0 + 0.3) = 10.4
  税后收入：10.4 × 0.5 = 5.2
  支出：2儿童 + 1老人 = 3.0
  净收入：2.2/年
```

### 2. 卡牌 ROI 对比
计算每张卡牌的投入产出比：

```
卡牌 | 消耗 | 期望收益 | 回收期 | 备注
献计 | 0金 | +0.5~0.8/年 | 即时 | 依赖智力
救治 | 5金 | +1💪 | 无直接回收 | 健康投资
投奔 | 3金 | 流民→职业 | 约4-6年 | 取决于新职业产出
```

### 3. 规模效应分析
检查涉及"按数量缩放"的机制是否存在线性最优解：

```
聚集：每流民+0.2 → N个流民总加成 = 0.2N
  对比：同成本下投奔的收益
  阈值：当流民数量 > X 时，聚集收益超过投奔
```

### 4. 参考基线对比
将新设计与已有体系对比：

```
战国（30%税率）基准：
  单人产出：1.3 × 0.3 = 0.39
  10人净收入：≈ 3.9 - 3.0 = 0.9

三国（50%税率）基准：
  单人产出：1.3 × 0.5 = 0.65
  10人净收入：≈ 5.2 - 3.0 = 2.2

结论：三国人均产出高，但战争消耗（事件/伤亡）需额外考虑
```

### 5. 极端场景分析
检查以下场景的数值合理性：
- 单人局：收入能否覆盖支出？存活周期多长？
- 满员局：经济是否溢出？有无消耗出口？
- 零金开局：是否可玩？卡牌消耗门槛是否合理？
