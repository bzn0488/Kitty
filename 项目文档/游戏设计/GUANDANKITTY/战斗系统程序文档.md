## 一、架构概览

### 核心设计原则

- **Agent 抽象**：不区分「玩家」和「敌人」，统一为 Agent。支持任意数量的 Agent（1v1 / 1v2 / 2v2 / Boss 战…）
- **修正器管道**：伤害计算和牌型检测支持外部注入修正器（遗物系统预留）
- **淘汰制回合**：Agent 依次出牌或 Pass，最后留下的 Agent 赢得回合

```
┌─────────────────────────────────────────────────────┐
│                    BattleManager                     │
│                  （战斗状态机总控）                    │
│                                                      │
│  ┌──────────────────────────────────────────────┐  │
│  │               RoundManager                    │  │
│  │          （回合轮次 + 淘汰判定）                │  │
│  │                                               │  │
│  │  Agents: [Player, Friendly, Enemy1, Enemy2]   │  │
│  │           ↑ 任意数量、任意类型                  │  │
│  └──────────────────────────────────────────────┘  │
│                                                      │
│  ┌──────────────┐  ┌──────────────┐                 │
│  │ ChainTracker │  │ DamageCalc   │                 │
│  │  接龙追踪     │  │  伤害计算     │                 │
│  │ (修正器管道)  │  │ (修正器管道)  │                 │
│  └──────────────┘  └──────────────┘                 │
│                                                      │
│  ┌──────────────────┐  ┌────────────────────────┐   │
│  │ SuppressionJudge │  │ CardPatternDetector    │   │
│  │   压制判定         │  │   牌型检测（修正器管道） │   │
│  └──────────────────┘  └────────────────────────┘   │
│                                                      │
│  ┌──────────────────────────────────────────────┐  │
│  │               CardRiver（牌河）                │  │
│  └──────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────┘
```

### 层级关系

```
BattleManager（总控）
 ├─ RoundManager（回合管理）
 │   └─ List<Agent>（参战方列表，按出牌顺序排列）
 │
 ├─ Agent（参战方）
 │   ├─ AgentType: Player / Friendly / Enemy
 │   ├─ List<HandZone>（手牌区，支持多手牌）
 │   ├─ StandardDeck?（仅 Player/Friendly 有牌堆）
 │   ├─ List<Monster>?（仅 Enemy 有怪物挂载）
 │   └─ 叫牌次数?（仅 Player）
 │
 ├─ CardRiver（牌河）
 ├─ ChainTracker（接龙追踪）
 ├─ DamageCalculator（伤害计算 + IDamageModifier 管道）
 ├─ CardPatternDetector（牌型检测 + IPatternModifier 管道）
 └─ SuppressionJudge（压制判定）
```

## 二、Agent（参战方抽象）

### 2.1 Agent 定义

不区分「玩家」和「敌人」——统一为 Agent。任意数量、任意类型。

```csharp
public class Agent
{
    public string Id { get; }
    public AgentType Type { get; }              // Player / Friendly / Enemy
    public List<HandZone> Hands { get; }        // 手牌区列表（支持多手牌：Boss战/双持）
    public StandardDeck? Deck { get; }          // Player/Friendly 有牌堆；Enemy 为 null
    
    // 仅在 Type == Player 时有效
    public int MaxCallCards { get; set; } = 3;
    public int RemainingCallCards { get; set; } = 3;
    public int CardsPerCall { get; set; } = 6;
    
    // 仅在 Type == Enemy 时有效
    public List<Monster>? Monsters { get; }     // 挂载的怪物列表
    
    // 战斗中的实时状态
    public bool HasPassed { get; set; }         // 本回合是否已 Pass（淘汰）
    public bool IsActive => !HasPassed;
    
    // 便捷查询
    public bool CanCallCards => Type == AgentType.Player && RemainingCallCards > 0;
    public bool IsPlayer => Type == AgentType.Player;
    public bool IsEnemy => Type == AgentType.Enemy;
    
    // 从任意手牌区中寻找能压制 target 的牌型
    // 返回 (handIndex, pattern) 或 null（无法压制）
    public (int handIndex, CardPattern pattern)? FindBestPlay(CardPattern target);
    
    // 对于 Enemy：遍历所有 HandZone，用最小合法牌型压制
    // 对于 Player/Friendly：不在此处决策，由 UI 层选择后传入
}

public enum AgentType { Player, Friendly, Enemy }
```

```csharp
    new Agent { Id = "boss", Type = AgentType.Enemy,

var agents = new List<Agent>
{
    new Agent { Id = "player", Type = AgentType.Player, ... },
    new Agent { Id = "ally", Type = AgentType.Friendly, ... },
    new Agent { Id = "enemy1", Type = AgentType.Enemy, ... },
    new Agent { Id = "enemy2", Type = AgentType.Enemy, ... }
};
```

> Agent 列表的顺序 = 出牌轮次顺序。BattleManager 按此顺序循环。

## 三、修正器管道（扩展性核心）

### 3.1 设计原理

遗物、贴纸、Boss 特殊规则等都可能需要修改伤害值或牌型判定。通过**接口 + 优先级管道**实现，核心逻辑零改动。

### 3.2 伤害修正器

```csharp
public interface IDamageModifier
{
    int Priority { get; }                        // 执行顺序（越小越先）
    int Modify(int damage, DamageContext ctx);    // 输入→输出，链式传递
}

public class DamageContext
{
    public CardPattern Pattern;       // 该手牌型
    public Agent Source;              // 出牌方
    public int BaseDamage;            // 修正前的基础伤害
    public int DepthMultiplier;       // 接龙深度倍率
    public bool IsWinningHand;        // 是否赢得回合
    public bool IsClearHand;          // 是否清空手牌
}

// DamageCalculator 改为实例，注入修正器
public class DamageCalculator
{
    private readonly List<IDamageModifier> _modifiers;
    
    public DamageCalculator(IEnumerable<IDamageModifier> modifiers = null)
    {
        _modifiers = modifiers?.OrderBy(m => m.Priority).ToList() ?? new();
    }
    
    public void AddModifier(IDamageModifier modifier) { ... }
    
    public int Calculate(DamageContext ctx)
    {
        int damage = ctx.BaseDamage;
        foreach (var mod in _modifiers)
            damage = mod.Modify(damage, ctx);
        return damage;
    }
}
```

### 3.3 牌型修正器

```csharp
public interface IPatternModifier
{
    int Priority { get; }
    // 返回 null = 不改变；返回新 Pattern = 替换
    CardPattern? Modify(CardPattern detected, List<Card> selectedCards, Agent source);
}

// CardPatternDetector 改为实例
public class CardPatternDetector
{
    private readonly List<IPatternModifier> _modifiers;
    
    public CardPattern? Detect(List<Card> cards, Agent source = null)
    {
        var pattern = DetectRaw(cards);
        if (pattern == null) return null;
        
        foreach (var mod in _modifiers)
        {
            var modified = mod.Modify(pattern, cards, source);
            if (modified != null) pattern = modified;
        }
        return pattern;
    }
    
    // 原始检测逻辑（不变）
    private static CardPattern? DetectRaw(List<Card> cards) { ... }
}
```

### 3.4 示例：遗物修正器

```csharp
// 遗物「三带一」：3张同数字 + 1张任意 → 视为三张牌型
public class ThreePlusOneModifier : IPatternModifier
{
    public int Priority => 100;
    
    public CardPattern? Modify(CardPattern detected, List<Card> cards, Agent source)
    {
        if (detected != null) return null; // 已是合法牌型，不修改
        if (cards.Count != 4) return null;
        
        // 检查是否是 3同数字 + 1任意
        var groups = cards.GroupBy(c => c.Rank).ToList();
        if (groups.Any(g => g.Count() == 3))
        {
            var triple = groups.First(g => g.Count() == 3);
            return new CardPattern(PatternType.Triple, triple.ToList(), triple.Key.GetHashCode());
        }
        return null;
    }
}

// 遗物「顺子强化」：6张以上顺子伤害×2
public class LongStraightDamageModifier : IDamageModifier
{
    public int Priority => 50;
    
    public int Modify(int damage, DamageContext ctx)
    {
        if (ctx.Pattern.Type == PatternType.Straight && ctx.Pattern.CardCount >= 6)
            return damage * 2;
        return damage;
    }
}
```

## 四、数据实体

### 4.1 Card（扑克牌）

```csharp
public class Card
{
    public Suit Suit { get; }        // ♠♣♥♦
    public Rank Rank { get; }        // 2~A
    public int FaceValue { get; }    // 牌面数值 2~14
    
    // 比较：用于同牌型大小比较
    public int CompareTo(Card other) => FaceValue.CompareTo(other.FaceValue);
}

public enum Suit { Spade, Club, Heart, Diamond }
public enum Rank { Two=2, Three, Four, Five, Six, Seven, Eight, Nine, Ten, Jack=11, Queen=12, King=13, Ace=14 }
```

> 牌不可变（immutable）。手牌区的牌是 Card 引用，牌堆中是 Card 实例。洗回牌堆时复用同一 Card 实例。

### 2.2 StandardDeck（标准牌堆）

```csharp
public class StandardDeck
{
    private List<Card> _cards;  // 当前牌堆（顶部 = index 0）
    
    public int Count { get; }
    
    public void Initialize();       // 创建52张牌并洗牌
    public Card Draw();             // 从顶部抽1张，牌堆减少
    public List<Card> Draw(int n);  // 抽n张
    public void ShuffleBack(List<Card> cards); // 将牌洗回牌堆（不改变牌堆中已有牌的顺序）
    public void RemovePermanently(List<Card> cards); // 永久移出（战败效果用）
}
```

> 不存在弃牌堆。牌河中的牌在回合结束时 `ShuffleBack`。被战败效果移出的牌从 `_cards` 中删除，不进入任何区域。

### 2.3 HandZone（手牌区）

```csharp
public class HandZone
{
    private List<Card> _cards;
    
    public int Count { get; }
    public IReadOnlyList<Card> Cards { get; }

    public void Add(Card card);             // 单张加入
    public void AddRange(List<Card> cards);  // 批量加入
    public List<Card> Remove(List<Card> cards); // 移除指定牌（打出/被弃）
    public List<Card> RemoveAll();           // 清空手牌

    public List<Card> FindAll(Func<Card, bool> predicate);

```

> 手牌无上限。玩家和敌方各持一个 HandZone 实例。敌方的 HandZone 对外只读可见。

### 2.4 CardPattern（牌型检测结果）

```csharp
public class CardPattern
{
    public PatternType Type { get; }    // 牌型
    public List<Card> Cards { get; }    // 构成该牌型的牌
    public int CardCount { get; }       // 出牌张数（= 伤害因子）
    public int CompareValue { get; }    // 用于同牌型比较的值
    
    // CompareValue 定义：
    //   单张/一对/三张/炸弹 → 牌的 FaceValue
    //   顺子/联队 → 最大牌的 FaceValue
}

public enum PatternType
{
    Single,     // 单张
    Pair,       // 一对
    Triple,     // 三张
    Straight,   // 顺子（≥5张连续）
    Tractor,    // 联队（≥3对连续）
    Bomb        // 炸弹（≥4张同数字）
}
```

### 2.5 CardPatternDetector（牌型检测器）

```csharp
public static class CardPatternDetector
{
    // 检测一组牌是否构成合法牌型，返回 CardPattern（含 CompareValue）
    // 若不合法，返回 null
    public static CardPattern Detect(List<Card> cards);
    
    // 私有方法
    private static bool IsSingle(List<Card> cards, out int compareValue);
    private static bool IsPair(List<Card> cards, out int compareValue);
    private static bool IsTriple(List<Card> cards, out int compareValue);
    private static bool IsStraight(List<Card> cards, out int compareValue);
    private static bool IsTractor(List<Card> cards, out int compareValue);
    private static bool IsBomb(List<Card> cards, out int compareValue);
}
```

> 检测优先级：从最严格到最宽松。优先检测炸弹→联队→顺子→三张→一对→单张。一旦匹配即返回，不回溯。

### 2.7 ChainTracker（接龙追踪器）

```csharp
public class ChainTracker
{
    public int PlayerHandCount { get; private set; }   // 本回合玩家第几次出牌（深度）
    public CardPattern LastPlayed { get; private set; } // 上一手出的牌型
    public Agent LastPlayedBy { get; private set; }     // 上一手是哪个 Agent 出的
    public List<Agent> PassedAgents { get; }             // 本回合已 Pass 的 Agent
    
    public void RecordPlay(CardPattern pattern, Agent agent);
    public void RecordPass(Agent agent);
    public void Reset();  // 回合开始时调用
    
    // 便捷查询
    public int CurrentDepth => PlayerHandCount;
    public int DepthMultiplier => 1 << (PlayerHandCount - 1);  // 2^(N-1)
    public bool AllEnemiesPassed(Agent player);  // 所有敌方 Agent 是否已 Pass
    public Agent? GetLastActiveAgent(Agent player); // 除玩家外最后一个未 Pass 的 Agent
}
```

> 深度 = 玩家出牌次数。其他 Agent 出牌不推进深度。Pass 只记录不推进深度。

### 2.9 DamageCalculator（伤害计算器）

```csharp
public class DamageCalculator
{
```

### 2.10 SuppressionJudge（压制判定器）

```csharp
public static class SuppressionJudge
```

## 三、怪物系统

### 3.1 Monster（怪物定义）

```csharp
public class Monster
{
    // 基础属性
    public string Id { get; }
    public string Name { get; }
    public int Level { get; }
    
    // HP
    public int BaseHP { get; }          // Lv1 基础血量
    public int CurrentHP => (int)(BaseHP * Math.Pow(1.1, Level - 1));
    
    // 牌池（骰子）
    public List<Card> CardPool { get; }  // 5张牌，可重复
    public int DrawsPerRound { get; }    // 每回合抽几张（默认1）
    
    // 战败效果
    public DefeatEffect DefeatEffect { get; }
}

public abstract class DefeatEffect
{
    public abstract string Description { get; }
    
    // 从玩家手牌中选择要移出的牌，返回移出的牌列表
    public abstract List<Card> SelectCardsToRemove(HandZone playerHand);
    
    // 移出数量是多少（补抽等量）
    public int RemoveCount(List<Card> selected) => selected.Count;
}
```

### 3.2 战败效果类型

```csharp
// 随机弃N张
public class RandomDiscardEffect : DefeatEffect
{
    public int Count { get; set; }
    public override List<Card> SelectCardsToRemove(HandZone hand)
        => hand.Cards.OrderBy(_ => Random).Take(Count).ToList();
}

// 弃掉所有 ≤N 的牌
public class DiscardBelowEffect : DefeatEffect
{
    public int Threshold { get; set; }
    public override List<Card> SelectCardsToRemove(HandZone hand)
        => hand.Cards.Where(c => c.FaceValue <= Threshold).ToList();
}

// 弃掉所有 ≥N 的牌
public class DiscardAboveEffect : DefeatEffect { ... }

// 弃掉最大N张
public class DiscardHighestEffect : DefeatEffect
{
    public int Count { get; set; }
    public override List<Card> SelectCardsToRemove(HandZone hand)
        => hand.Cards.OrderByDescending(c => c.FaceValue).Take(Count).ToList();
}

// 弃掉所有某花色的牌
public class DiscardSuitEffect : DefeatEffect
{
    public Suit TargetSuit { get; set; }
    public override List<Card> SelectCardsToRemove(HandZone hand)
        => hand.Cards.Where(c => c.Suit == TargetSuit).ToList();
}

// 弃掉所有偶数牌
public class DiscardEvenEffect : DefeatEffect { ... }

// 弃掉所有奇数牌
public class DiscardOddEffect : DefeatEffect { ... }
```

### 3.3 DefeatEffectExecutor（战败效果执行器）

```csharp
public static class DefeatEffectExecutor
{
    // 执行所有怪物的战败效果（全部触发）
    // 返回 (移出的牌列表, 补抽的牌列表)
    public static (List<Card> removed, List<Card> drawn) Execute(
        List<Monster> monsters,
        HandZone playerHand,
        StandardDeck playerDeck
    );
    
    // 流程：
    // 1. 对每个怪物依次执行 SelectCardsToRemove
    // 2. 从 playerHand 中移出选中的牌
    // 3. 从 playerDeck 中永久删除这些牌
    // 4. 从 playerDeck 中抽等量新牌加入 playerHand
}
```

### 3.4 EnemyAgent（敌方 AI）

```csharp
public class EnemyAgent
{
    public HandZone Hand { get; }
    public List<Monster> ActiveMonsters { get; }
    
    // 回合结束时，从所有怪物牌池抽牌
    public void DrawCardsForRound()
    {
        foreach (var monster in ActiveMonsters)
            for (int i = 0; i < monster.DrawsPerRound; i++)
                Hand.Add(monster.CardPool[Random.Range(0, monster.CardPool.Count)]);
                // 注：抽的是复制品——牌池不消耗，Card 是新的实例但值相同
    }
    
    // AI 出牌决策：尝试用最小合法牌型压制
    // 返回 null 表示 Pass
    public CardPattern DecidePlay(CardPattern playerHand);
    
    // 实现逻辑：
    // 1. 确定需要压制的牌型（playerHand.Type）
    // 2. 若 playerHand 是炸弹 → 在手牌中找更大的炸弹，没有则 Pass
    // 3. 否则：
    //    a. 在手牌中找同牌型 + 更大 CompareValue 的牌
    //    b. 从所有满足条件的牌型中，选 CompareValue 最小的
    //    c. 若同牌型找不到，检查是否有炸弹（炸弹可跨牌型）
    //    d. 都没有 → Pass
}
```

> 关键行为：「用最小合法牌型压制」。不拆牌 = 检测到一组可压制的牌后直接打出，不对该组牌做"是否拆散更好牌型"的判断。

## 五、战斗状态机（淘汰制多 Agent 回合）

### 5.1 回合规则

```
Agent 列表按顺序出牌。每回合开始时，所有 Agent 均为「活跃」状态。
轮到某 Agent 时：
  · 可以出牌（压制上一手）→ 记录，轮到下一个 Agent
  · 可以 Pass → 该 Agent 标记为「淘汰」（本回合不再参与）
  · 如果所有非玩家 Agent 都已淘汰 → 玩家可自压或 Pass
  · 如果玩家 Pass → 最后一个活跃的 Agent 赢得回合
  · 玩家清空手牌 → 立即赢得回合（不等其他 Agent 回应）
```

### 5.3 BattleManager 伪代码

```csharp
public class BattleManager
{
    public enum State { Init, RoundStart, AgentTurn, RoundSettlement, 
                        RoundEnd, Victory, Defeat }
    
    public State CurrentState { get; private set; }
    
    private List<Agent> _agents;          // 所有参战方（按出牌顺序）
    private int _currentAgentIndex;       // 当前轮到的 Agent
    private CardRiver _river;
    private ChainTracker _chain;
    private DamageCalculator _damageCalc;
    private CardPatternDetector _patternDetector;
    
    private Agent PlayerAgent => _agents.First(a => a.IsPlayer);
    private IEnumerable<Agent> EnemyAgents => _agents.Where(a => a.IsEnemy);
    private int TotalEnemyHP => EnemyAgents.Sum(a => a.Monsters?.Sum(m => m.CurrentHP) ?? 0);
    
    // === 生命周期 ===
    
    public void StartBattle(List<Agent> agents)
    {
        CurrentState = State.Init;
        _agents = agents;
        
        // 初始化所有 Agent
        foreach (var agent in _agents)
        {
            if (agent.Deck != null)
            {
                agent.Deck.Initialize();
                agent.Hands[0].AddRange(agent.Deck.Draw(8)); // 初始8张
            }
        }
        
        BeginRound();
    }
    
    private void BeginRound()
    {
        CurrentState = State.RoundStart;
        _river.Clear();
        _chain.Reset();
        
        // 全员标记为活跃
        foreach (var agent in _agents)
            agent.HasPassed = false;
        
        // 触发"回合开始"效果（预留）
        
        // 从第一个 Agent 开始
        _currentAgentIndex = 0;
        AdvanceToNextAgent();
    }
    
    // === 轮询核心 ===
    
    private void AdvanceToNextAgent()
    {
        // 跳过已淘汰的 Agent
        while (_currentAgentIndex < _agents.Count && _agents[_currentAgentIndex].HasPassed)
            _currentAgentIndex++;
        
        // 轮完一圈 → 回到开头
        if (_currentAgentIndex >= _agents.Count)
            _currentAgentIndex = 0;
        
        // 再次跳过已淘汰的
        while (_currentAgentIndex < _agents.Count && _agents[_currentAgentIndex].HasPassed)
            _currentAgentIndex++;
        
        // 检查是否只剩一个活跃 Agent
        var activeAgents = _agents.Where(a => a.IsActive).ToList();
        
        if (activeAgents.Count == 1)
        {
            var lastAgent = activeAgents[0];
            if (lastAgent.IsPlayer)
            {
                // 其他人都 Pass 了，玩家可以自压或 Pass
                // 注：此时 lastAgent 仍处于活跃状态，但 LastPlayed 可能是自己的牌
                CurrentState = State.AgentTurn;
                // 等待玩家输入（自压 或 Pass）
            }
            else
            {
                // 玩家已 Pass，敌方是最后一个活跃 → 敌人赢
                SettleRound(winner: lastAgent);
            }
            return;
        }
        
        if (activeAgents.Count == 0)
        {
            // 理论上不会发生（至少玩家要 Pass 才会触发结算）
            return;
        }
        
        var currentAgent = _agents[_currentAgentIndex];
        CurrentState = State.AgentTurn;
        
        if (currentAgent.IsPlayer)
        {
            // 等待玩家输入
        }
        else
        {
            // AI 自动决策
            ProcessAgentTurn(currentAgent);
        }
    }
    
    // === 玩家行动（由 UI 层调用） ===
    
    public void PlayerPlay(List<Card> selectedCards)
    {
        var player = PlayerAgent;
        if (CurrentState != State.AgentTurn || !player.IsActive) return;
        
        var pattern = _patternDetector.Detect(selectedCards, player);
        if (pattern == null) return; // 非法牌型
        
        // 压制判定（含自压）
        if (_chain.LastPlayed != null)
        {
            if (!SuppressionJudge.CanSuppress(pattern, _chain.LastPlayed))
                return;
        }
        
        // 从手牌移除
        player.Hands[0].Remove(selectedCards);
        
        // 即时伤害
        bool isClearHand = player.Hands.All(h => h.IsEmpty);
        int damage = _damageCalc.Calculate(new DamageContext
        {
            Pattern = pattern,
            Source = player,
            BaseDamage = pattern.Cards.Sum(c => c.FaceValue) * pattern.CardCount * _chain.DepthMultiplier,
            DepthMultiplier = _chain.DepthMultiplier,
            IsWinningHand = false,   // 暂时未知，结算时修正
            IsClearHand = isClearHand
        });
        ApplyDamage(damage);
        
        // 记录
        _river.Add(pattern, player);
        _chain.RecordPlay(pattern, player);
        
        if (isClearHand)
        {
            // 清空手牌 → 立即获胜 → 补抽8张
            player.Hands[0].AddRange(player.Deck.Draw(8));
            SettleRound(winner: player);
            return;
        }
        
        // 轮到下一个 Agent
        _currentAgentIndex++;
        AdvanceToNextAgent();
    }
    
    public void PlayerPass()
    {
        var player = PlayerAgent;
        if (CurrentState != State.AgentTurn || !player.IsActive) return;
        
        player.HasPassed = true;
        _chain.RecordPass(player);
        
        _currentAgentIndex++;
        AdvanceToNextAgent();
    }
    
    public void PlayerCallCards()
    {
        var player = PlayerAgent;
        if (CurrentState != State.AgentTurn || !player.IsActive || !player.CanCallCards) return;
        
        player.Hands[0].AddRange(player.Deck.Draw(player.CardsPerCall));
        player.RemainingCallCards--;
        // 仍处于 AgentTurn，等待出牌或 Pass
    }
    
    // === 非玩家 Agent 自动决策 ===
    
    private void ProcessAgentTurn(Agent agent)
    {
        if (_chain.LastPlayed == null)
        {
            // 回合首手只有玩家能出（玩家总是先手）
            // 非玩家 Agent 在第一轮之前不会到这里
            agent.HasPassed = true;
            _chain.RecordPass(agent);
        }
        else
        {
            var (handIdx, pattern) = agent.FindBestPlay(_chain.LastPlayed) ?? (-1, null);
            
            if (pattern != null)
            {
                // 出牌（不造成伤害，牌进牌河）
                agent.Hands[handIdx].Remove(pattern.Cards);
                _river.Add(pattern, agent);
                _chain.RecordPlay(pattern, agent);
            }
            else
            {
                // Pass
                agent.HasPassed = true;
                _chain.RecordPass(agent);
            }
        }
        
        _currentAgentIndex++;
        AdvanceToNextAgent();
    }
    
    // === 回合结算 ===
    
    private void SettleRound(Agent winner)
    {
        CurrentState = State.RoundSettlement;
        
        if (winner.IsPlayer)
        {
            // 玩家阵营赢
            // 修正最后一手伤害（赢得回合 ×2）
            // 赢家通吃效果结算（预留）
            
            if (TotalEnemyHP <= 0)
            {
                CurrentState = State.Victory;
                return;
            }
        }
        else
        {
            // 敌方阵营赢 → 战败效果（所有敌方怪物效果全部触发）
            foreach (var enemy in EnemyAgents)
            {
                if (enemy.Monsters == null) continue;
                DefeatEffectExecutor.Execute(enemy.Monsters, PlayerAgent.Hands[0], PlayerAgent.Deck);
            }
            
            if (PlayerAgent.Deck.IsEmpty && PlayerAgent.Hands.All(h => h.IsEmpty))
            {
                CurrentState = State.Defeat;
                return;
            }
        }
        
        EndRound();
    }
    
    private void EndRound()
    {
        CurrentState = State.RoundEnd;
        
        // 牌河洗回
        PlayerAgent.Deck.ShuffleBack(_river.GetAllCards());
        _river.Clear();
        
        // 敌方成长
        foreach (var enemy in EnemyAgents)
        {
            foreach (var monster in enemy.Monsters ?? new())
            {
                for (int i = 0; i < monster.DrawsPerRound; i++)
                    enemy.Hands[0].Add(monster.CardPool[Random.Range(0, monster.CardPool.Count)]);
            }
        }
        
        BeginRound();
    }
}
```

## 六、数据流

### 6.1 出牌伤害流程

```
玩家选牌
  ↓
CardPatternDetector.Detect(cards, agent) → CardPattern（含修正器管道）
  ↓
SuppressionJudge.CanSuppress(pattern, chain.LastPlayed) → bool
  ↓ (通过)
Agent.Hands[handIdx].Remove(cards)
  ↓
DamageCalculator.Calculate(DamageContext) → int damage（含修正器管道）
  ↓
扣减敌方 HP
  ↓
CardRiver.Add(pattern, agent)
  ↓
ChainTracker.RecordPlay(pattern, agent)
  ↓
清空手牌?
  ├─ 是 → ×10 + 补抽8张 + 立即获胜结算
  └─ 否 → AdvanceToNextAgent（下一个 Agent 轮次）
```

### 6.2 淘汰制轮询流程

```
AdvanceToNextAgent()
  ↓
跳过所有 HasPassed=true 的 Agent
  ↓
活跃 Agent 数量?
  ├─ 1个 → 是玩家? 
  │          ├─ 是 → 等待玩家输入（自压 或 Pass → 玩家赢）
  │          └─ 否 → 该Agent赢得回合 → SettleRound
  ├─ 0个 → （不会发生）
  └─ 多个 → 当前Agent轮次
              ├─ 非玩家 → AI自动决策（出牌/Pass）
              └─ 玩家 → 等待UI输入
```

### 6.3 回合结算流程

```
SettleRound(winner)
  ↓
玩家阵营赢?
  ├─ 是 → 赢家通吃结算 + 检查敌人HP≤0 → Victory
  └─ 否 → 战败效果（所有敌方怪物全部触发）
            ↓
          检查玩家牌堆+手牌=0 → Defeat
  ↓
未结束 → EndRound:
  Deck.ShuffleBack(牌河所有牌)
  所有 Enemy 的怪物牌池抽牌
  ↓
BeginRound → 下一回合
```

## 七、关键数据结构总结

```
Card                    — 不可变牌数据（Suit, Rank, FaceValue）
StandardDeck            — 52张牌堆，Draw/ShuffleBack/RemovePermanently
HandZone                — 手牌区，Add/Remove/FindAll
CardPattern             — 牌型检测结果（Type, Cards, CardCount, CompareValue）
CardPatternDetector     — 实例类，Detect + IPatternModifier 管道
SuppressionJudge        — 静态类，CanSuppress(hand, target) → bool（与Agent身份无关）
ChainTracker            — 接龙状态（PlayerHandCount, LastPlayed, LastPlayedBy, PassedAgents）
CardRiver               — 牌河（List<RiverEntry> 含 Agent 引用）
DamageCalculator        — 实例类，Calculate + IDamageModifier 管道

Agent                   — 参战方抽象（Type, Hands[], Deck?, Monsters?, HasPassed）
AgentType               — Player / Friendly / Enemy

Monster                 — 怪物数据（BaseHP, Level, CardPool, DrawsPerRound, DefeatEffect）
DefeatEffect (abstract) — 战败效果基类
DefeatEffectExecutor    — 静态类，Execute(monsters, hand, deck)

IDamageModifier         — 伤害修正器接口（遗物/贴纸预留）
IPatternModifier        — 牌型修正器接口（遗物/贴纸预留）

BattleManager           — 战斗状态机总控（淘汰制多Agent回合）
```
