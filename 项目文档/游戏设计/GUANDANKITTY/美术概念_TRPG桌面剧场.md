# 美术概念：TRPG 桌面剧场

> 版本：v0.1 | 创建：2026-07-06
> 状态：草案

---

## 一、核心概念

GuandanKitty 的世界发生在一张 TRPG 牌桌上。你和戴着神秘面具的 DM（地下城主）进行生死斗牌，通过他设下的层层考验。

**设计原则：一切皆桌上物件。** 不出现脱离桌面场景的 UI 面板——所有游戏元素都是你放在桌上的东西。

---

## 二、物件映射

### 2.1 核心物件

| 桌上物件 | 对应系统 | 视觉描述 |
|------|------|------|
| 🃏 **扑克牌** | 玩家手牌、敌方手牌 | 标准扑克，俯视/3D 持握 |
| 🃏 **怪物卡** | 怪物 | 独立卡牌，角标显示等级点数（如 ♠♠♠ = Lv3） |
| 🃏 **遗物卡** | 遗物 | 特殊边框卡牌，立在迷你支架上 |
| 🎴 **贴纸** | 卡牌贴纸 | 物理贴纸效果——粘在牌面上，有厚度感 |
| 🎲 **骰子** | 怪物牌池 | 每个怪物身边放置对应骰子，每回合投掷 |
| 🎲 **d100** | 随机事件 | DM 投骰子宣告结果 |
| 🪙 **筹码** | 金币 | 堆叠在玩家侧，不同颜色 = 不同面额 |
| 🩸 **血滴标记** | HP | 红色筹码或血滴状标记，放在 DM 侧 |

### 2.2 场景物件

| 物件 | 说明 |
|------|------|
| 🎭 **DM 屏风** | 分隔 DM 与玩家区域，DM 侧敌方手牌**亮着**（对玩家可见） |
| 📐 **战斗网格** | 桌面中央的网格区域 = 牌河，牌打在此处 |
| 📜 **角色卡** | 玩家侧放置，可翻开（叫牌 = 翻开新一页） |
| 🃏 **命运牌堆** | DM 侧一副特殊牌，用于随机事件判定 |

## 三、怪物卡设计

怪物以扑克牌形式呈现，与传统扑克共享视觉语言：

```
┌──────────────┐
│ ♠ 哥布林     │  ← 名称
│              │
│   👹         │  ← 插画
│              │
│ Lv ♠♠♠      │  ← 等级（点数表示）
│              │
│ HP 605       │  ← HP 贡献
│ 🎲 x1/回合   │  ← 抽牌速率
│ 弃3补3       │  ← 战败效果
│ [2,3,3,4,5]  │  ← 牌池
└──────────────┘
```

- 花色 = 怪物稀有度（♠普通 ♥精英 ♦稀有 ♣传说）
- 角标点数 = 等级（1♠ = Lv1, 3♠ = Lv3）
- 等级提升时角标点数增加，视觉上有「升级」感

## 四、骰子系统

### 怪物骰子

每个怪物身边放置对应骰子。每回合结束时：

- 怪物骰子自动投掷 → 从牌池中对应取出牌 → 加入敌方手牌
- 投骰动画 = 随机性的视觉包装（而非冰冷的后台计算）

### DM 骰子（d100）

用于随机事件判定。DM 投出 d100，玩家看到骰子翻滚——仪式感本身即是正反馈的信号包装。

## 五、场景氛围

| 元素 | 方向 |
|------|------|
| 桌面 | 深色木纹或绿色桌毡（felt） |
| 灯光 | 顶光吊灯，偏暖色，投下圆形光斑 |
| DM | 半隐藏于屏风后，仅露手和面具，神秘感 |
| 音效 | 牌摩擦声、骰子撞击声、筹码堆叠声、DM 低语 |
| 节奏 | 出牌→骰子→DM 冷笑/赞叹——回合间有微妙的"表演停顿" |
