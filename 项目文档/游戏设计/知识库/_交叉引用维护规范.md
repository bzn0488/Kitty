# 交叉引用索引维护规范

> 知识整理中的内容更新时，必须同步更新 `_交叉引用索引.md`。

## 一、何时必须更新

| 场景 | 动作 | 优先级 |
|------|------|:------:|
| 新增一个知识文件 | 在交叉引用索引中新增一行 | 强制 |
| 在已有文件中新增了一个章节，该章节引用了其他文件 | 检查并更新该文件对应的交叉引用行 | 强制 |
| 删除了一个知识文件 | 删除交叉引用索引中对应的行 | 强制 |
| 修改了一个文件中的概念定义，导致与其他文件的引用关系变化 | 更新交叉引用索引中的说明字段 | 建议 |

## 二、检查清单

更新前，问自己三个问题：

1. **这个新增/修改的内容引用了知识整理中的哪个文件？**
   - 例："我在 03 新增了关于 蔡格尼克效应 的应用"→ 引用 02
   
2. **这个引用关系是新的吗？还是已有的？**
   - 新的 → 新增行
   - 已有的 → 检查说明是否需要更新

3. **这个内容被其他文件引用了吗？**
   - 如果有，需要在其他文件的交叉引用行中也补充

## 三、写入格式

`_交叉引用索引.md` 中的每一行按此格式：

```
| 来源 | → 关联文件 | 关联概念 | 用途 |
```

| 字段 | 填写规则 | 示例 |
|------|----------|------|
| **来源** | 文件名（不含 .md） | `02-玩家心理学` |
| **→ 关联文件** | 被引用的文件名（不含 .md） | `03-游戏机制` |
| **关联概念** | 具体的概念名称，多个用逗号分隔 | `蔡格尼克效应, 强制循环` |
| **用途** | 一句话说明引用目的 | 02定义心理学原理，03展开为设计技法 |

## 四、依赖链同步

交叉引用索引底部的**依赖链图**也需要同步更新。

```
当前依赖链：
02(心理基础) → 03(机制技法) → 04(节奏心流) → 05(叙事设计)
                                              ↑
01(核心概念) → 03(机制技法) ------------------┘
     ↓
04(节奏心流)
```

如果新增的文件改变了依赖关系，调整这个链图。

## 五、过期检测

当以下迹象出现时，交叉引用索引可能已过期：
- 知识整理中某个文件有 **显式引用语句**（如"详见 02-玩家心理学"），但交叉引用索引中没有对应行
- 删除或合并了文件，但没有同步更新索引
- 发现索引中的文件数量与实际文件数不匹配

**修复方式**：重新扫描所有知识文件中的引用语句，生成新的交叉引用矩阵。
