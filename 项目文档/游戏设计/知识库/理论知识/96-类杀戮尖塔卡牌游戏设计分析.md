# 类杀戮尖塔卡牌游戏设计分析

> 调研目标：超越单一游戏分析，系统对比类杀戮尖塔（StS-like）卡牌Roguelike游戏的
> 设计创新和差异化策略，提炼对荒野与曙光和家族纪元的可借鉴设计模式。

---

## 一、类杀戮尖塔品类的定义与演进

### 1.1 品类的核心特征

类杀戮尖塔（StS-like）游戏共享以下核心特征：

| #   | 特征                | 说明                                   |
| --- | ------------------- | -------------------------------------- |
| 1   | **DBG（牌库构筑）** | 战斗中获得新牌，边打边构筑，非赛前组卡 |
| 2   | **Roguelike进程**   | 随机地图、永久死亡、重玩驱动           |
| 3   | **单局1小时内**     | 完整的一局游戏不超过1小时              |
| 4   | **卡牌驱动战斗**    | 卡牌是主要的交互和决策手段             |
| 5   | **三选一奖励**      | 战后从3张卡中选择1张加入牌库           |
| 6   | **意图系统**        | 怪物行动对玩家可见（或部分可见）       |

### 1.2 品类演进树

```
杀戮尖塔 (2019) ──┬── 奠基者：确立了品类标准
                  │
                  ├── Monster Train (2020) ── 多线防守 + 种族协同 + 契约系统
                  │
                  ├── 欺诈之地/Griftlands (2021) ── 双牌库 + 叙事驱动 + 伙伴系统
                  │
                  ├── Roguebook (2021) ── 双角色 + 墨水系统 + 自动协同
                  │
                  ├── Wildfrost (2022) ── 计时器 + 物品系统 + 极简卡牌
                  │
                  ├── 背包英雄/Backpack Hero (2022) ── 背包空间管理 + 物品交互
                  │
                  ├── Across the Obelisk (2021) ── 4人组队 + 实时联机 + 天赋树
                  │
                  └── Astrea (2023) ── 六面骰替代卡牌 + 双色净化系统
```

## 二、主要作品的设计创新对比

### 2.1 核心机制创新矩阵

| 游戏              | 核心创新      | 费用系统                | 构筑维度             | 战斗模式    | 独特约束         |
| ----------------- | ------------- | ----------------------- | -------------------- | ----------- | ---------------- |
| **杀戮尖塔**      | 品类奠基      | 3费/回合固定            | 卡牌+遗物            | 1v1意图     | 删牌经济         |
| **Monster Train** | 多线防守      | 3费/回合固定            | 卡牌+契约+种族       | 3线防御     | 契约升级可选     |
| **欺诈之地**      | 双牌库系统    | 行动点（协商/战斗独立） | 卡牌+伙伴+关系       | 1v1+伙伴    | 关系影响卡牌获取 |
| **Roguebook**     | 双角色+墨水   | 每回合+1（共享）        | 双角色卡牌+遗物+墨水 | 1v1+双角色  | 墨水解锁地图     |
| **Wildfrost**     | 计时器+物品   | 1行动/回合+时钟限制     | 卡牌+物品+伴侣       | 多怪+时间轴 | 时钟计数器       |
| **背包英雄**      | 空间管理+物品 | 无传统费用              | 物品+背包空间        | 自动攻击    | 网格空间管理     |
| **Astrea**        | 骰子替代卡牌  | 骰子面消耗              | 骰子+净化/腐化       | 1v1意图     | 骰子运气管理     |

### 2.2 Monster Train：多线防守的创新

```
核心差异：
  杀戮尖塔：你是攻击者，主动去打怪物
  Monster Train：你是防守者，怪物来打你

设计影响：
  - "防守者心态"改变了卡牌价值评估——防御卡价值天然更高
  - 多层防御 → 需要管理多个战线的资源分配
  - 契约系统（每次胜利后可选加强怪物或加强自己）→ 动态难度自选

关键设计创新：
  1. 多线战斗（3层）+ 单位部署 → 位置策略
  2. 种族协同（地狱/觉醒/冥卫/熔体） → 清晰构筑引导
  3. 契约系统 → 玩家自选难度曲线
  4. "Boss战在顶层，必须在底层前清理" → 创造时间压力

对荒野的参考：多线监控的资源分配、Boss在限时前的压力感
```

### 2.3 欺诈之地/Griftlands：叙事+双牌库

```
核心差异：
  杀戮尖塔：纯粹的卡牌战斗
  欺诈之地：卡牌战斗 + 卡牌协商（两个独立牌库）

设计影响：
  - 双牌库意味着双倍构筑深度——战斗牌和协商牌互不通用
  - 叙事驱动——角色关系和故事走向影响卡牌获取
  - 伙伴系统——NPC伙伴有独立牌库，增加队伍维度

关键设计创新：
  1. 协商系统 → 用卡牌进行辩论/说服/威胁，替代传统对话树
  2. 关系系统 → NPC好感度影响可获得的卡牌和伙伴
  3. 伙伴独立牌库 → 多角色卡牌管理的先驱
  4. 三天循环 → Roguelike进度与叙事的结合

对家族纪元的参考：卡牌作为社交/交涉手段（而非纯战斗工具）、关系影响卡牌获取
```

### 2.4 Roguebook：双角色协同

```
核心差异：
  杀戮尖塔：单人
  Roguebook：双角色同时在场，各自独立牌库

设计影响：
  - 角色间自动协同——某些卡牌效果受另一角色状态影响
  - 墨水系统（探索地图的资源）→ 替代传统路线选择的资源约束
  - 遗物锻造 → 可控的遗物系统（非纯随机）

关键设计创新：
  1. 双角色独立牌库+共享费用 → 多角色卡牌分配决策
  2. 墨水作为探索资源 → 探索与卡牌构筑挂钩
  3. 自动协同卡牌 → 降低协同发现难度

对荒野的参考：双角色+独立牌库+共享费用的管理模型
```

### 2.5 Wildfrost：时间轴+极简设计

```
核心差异：
  杀戮尖塔：回合制
  Wildfrost：计数器时间轴 + 极简卡牌（每角色5卡上限）

设计影响：
  - 计时器取代回合 → 行动不是"每回合X费"，而是"每X秒/每X计数器"
  - 卡牌上限极低（卡组通常10-15张） → 构筑更精确
  - 伴侣系统 → 卡牌与场上角色的绑定关系

关键设计创新：
  1. 计时器作为核心节奏 → 行动频率受计数器管理
  2. 极简牌库 → "牌库精简是美德"的极致化
  3. 物品+角色+卡牌三重构筑 → 装备与卡牌的交互

对荒野的参考：时间轴设计的先行者、极简牌库的可行性
```

## 三、超越杀戮尖塔的设计创新

### 3.1 费用系统的创新

| 游戏          | 费用系统             | 创新点                            | 对荒野的参考价值 |
| ------------- | -------------------- | --------------------------------- | ---------------- |
| 杀戮尖塔      | 3费/回合固定         | 基准                              | —                |
| Monster Train | 3费/回合+升级        | 费用升级是重要成长维度            | 🔴 低（体系不同） |
| 欺诈之地      | 行动点（3-4点/回合） | 行动点=任意搭配（无卡牌费用概念） | 🟡 中             |
| Roguebook     | 共享费用（每回合+1） | 双角色共享费用创造分配决策        | 🔴 **最高**       |
| Wildfrost     | 每回合1行动+时钟     | 极简费用+外部计时器               | 🟡 中             |
| 背包英雄      | 无传统费用           | 空间=费用                         | 🟢 低             |

**Roguebook的费用模式对荒野的参考价值最高**：双角色共享费用 + 每回合增长，与荒野的"共享时间轴+多角色"模型最接近。

### 3.2 构筑深度的扩展

| 游戏          | 构筑维度数 | 维度构成                | 协同密度 |
| ------------- | ---------- | ----------------------- | -------- |
| 杀戮尖塔      | 2          | 卡牌+遗物               | 中       |
| Monster Train | 4          | 卡牌+遗物+契约+种族升级 | 高       |
| 欺诈之地      | 3          | 卡牌+伙伴+关系          | 中高     |
| Roguebook     | 4          | 双角色牌+遗物+墨水+锻造 | 高       |
| Wildfrost     | 3          | 卡牌+物品+伴侣          | 中       |

**趋势**：类StS游戏的发展方向是**增加构筑维度**，而非增加卡牌数量。更多的构筑维度 = 更丰富的协同可能 = 更高的重玩性。

### 3.3 失败与重试设计的创新

| 游戏          | 失败惩罚 | 跨局继承        | 鼓励重试的方式 |
| ------------- | -------- | --------------- | -------------- |
| 杀戮尖塔      | 全丢     | 解锁新卡/新角色 | 进阶系统       |
| Monster Train | 全丢     | 解锁契约等级    | 契约自选难度   |
| 欺诈之地      | 部分保留 | 关系+故事进度   | 叙事驱动       |
| Roguebook     | 全丢     | 解锁墨水颜色    | 不同角色组合   |
| 背包英雄      | 全丢     | 解锁新物品      | 不同的背包策略 |

## 四、类杀戮尖塔游戏的通用设计原则

### 4.1 卡牌经济的共性规律

尽管各游戏费用系统不同，但卡牌经济存在共性规律：

```
好卡的特征：
  ✅ 费用低但价值高 → 直觉可感知的"超模"
  ✅ 在特定构筑中价值暴增 → 协同驱动
  ✅ 无严格替代品 → 每张卡有独特定位

坏卡的特征：
  ❌ 费用高但效果不确定 → 风险与回报不匹配
  ❌ 在任何构筑中都不出彩 → 缺少Build信号
  ❌ 有严格上位替代 → "这张卡有何存在意义？"
```

### 4.2 DBG构筑节奏的共性

```
前1/3（确立期）：
  - 拿基础卡 → 确立初步方向
  - 目标：费用曲线合理，没有明显短板

中间1/3（成长期）：
  - 拿核心协同卡 → 构筑自洽
  - 目标：弃掉初始卡，核心协同已成型

后1/3（精炼期）：
  - 删牌/升级 → 构筑极致化
  - 目标：针对Boss调整，构筑无可替换卡
```

### 4.3 类StS与TCG的设计差异

| 维度       | TCG（如万智牌/炉石） | 类StS（DBG+Roguelike）     |
| ---------- | -------------------- | -------------------------- |
| 牌库来源   | 赛前组卡             | 赛中积累                   |
| 费用曲线   | 每回合+1，上限10     | 固定（3费），或独特体系    |
| 卡牌平衡   | 多对多（天梯环境）   | 单局内闭环（局内平衡）     |
| 运气因素   | 抽牌运气             | 抽牌+卡牌掉落+遗物随机     |
| 重玩驱动力 | 天梯排名/卡牌收集    | 不同构筑可能性             |
| 失败惩罚   | 掉分/掉段            | 全丢从零开始（或部分继承） |

**关键洞察**：类StS的卡牌平衡不需要像TCG那样考虑全局环境，只需要确保"每局游戏内，不同构筑路线都能到达终点"。这大大降低了平衡难度，但增加了"每局可重玩性"的设计要求。

## 五、对荒野与曙光可直接借鉴的设计

### 5.1 Monster Train的多线资源分配 → 荒野的多角色管理

Monster Train需要玩家管理3条战线的单位部署和资源分配。这直接对应荒野的多角色管理：

```
借鉴点：
  - 多线/多角色的资源分配决策
  - "哪条线（哪个角色）需要更多资源"的实时判断
  - 不同角色（线）之间的支援机制
```

### 5.2 Roguebook的双角色共享费用 → 荒野的共享时间轴

Roguebook的双角色共享每回合增长的资源（费用），每回合玩家需决定"让谁行动"。

```
借鉴点：
  - 共享资源池下角色间的资源分配
  - 双角色卡牌的差异化定位（攻/守/辅）
  - 行动顺序的选择权交给玩家
```

### 5.3 Wildfrost的计时器模式 → 荒野的时间轴

Wildfrost的时钟计数器是最接近荒野时间轴的设计（在类StS中）。

```
借鉴点：
  - 计数器的视觉呈现方式
  - 极简牌库与时间轴节奏的匹配
  - "等待也是策略"的设计理念
```

### 5.4 欺诈之地的双牌库 → 荒野的复合玩法

欺诈之地的战斗+协商双牌库证明了一个游戏中可以有两个独立的卡牌系统。

```
借鉴点：
  - 荒野的外线（探索）+内线（战斗）是否需要两套卡牌？
  - 如果两套卡牌：一套用于探索决策，一套用于战斗
  - 两套卡牌之间的资源互通
```

## 六、类杀戮尖塔游戏的局限性

### 6.1 品类天花板

| 局限性           | 说明                       | 突破方向                     |
| ---------------- | -------------------------- | ---------------------------- |
| **战斗同质化**   | 多数类StS只是"换皮+微创新" | 野外的时间轴模型本身就是突破 |
| **构筑引导不足** | 新手不知道什么卡配什么卡   | 明确的Build信号系统          |
| **剧情薄弱**     | Roguelike与叙事的矛盾      | 欺诈之地的叙事融合是方向     |
| **视觉疲劳**     | 卡牌战斗的视觉反馈有限     | 部位破坏的视觉反馈是机会     |
| **学习曲线陡峭** | 卡牌游戏的通病             | 渐进式难度+教学关卡          |

### 6.2 荒野与曙光的差异化机会

```
品类现状 → 荒野的突破点：

类StS战斗是"回合制" → 野外的"时间轴"是品类创新
类StS是"单人" → 野外的"多角色"是差异化
类StS探索是"选路" → 野外的"领地建设"是融合创新
类StS构筑是"纯卡牌" → 野外的"卡牌+角色+领地"是多维度成长
```
