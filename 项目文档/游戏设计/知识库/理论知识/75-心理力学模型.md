# 心理力学模型（Psychomechanics）

> 标签：`player-psychology`, `motivation`, `behavior-change`
> 关联概念：`21-驱动力模型.md`, `46-正反馈与负反馈循环.md`, `48-自我决定理论SDT.md`
> 来源：动机心理学（Motivational Psychology）— 行为改变的综合理论视角
> 完整展开见：`游戏设计理论/02-玩家心理学.md §三`（Fogg模型）、`游戏设计理论/03-游戏机制设计技法.md §1.11`（强制循环）

---

## 定义与范畴

**心理力学模型**（Psychomechanics）来源于动机心理学，描述**人是否会改变行为**的核心机制。它与驱动力模型紧密关联但视角不同：

| 模型 | 核心问题 | 关注点 |
|------|----------|--------|
| **驱动力模型（水桶理论）** | "选哪个行为？" | 行为之间的比较选择 |
| **心理力学模型** | "**是否改变**当前行为？" | 行为改变的动力与阻力博弈 |

> 如果说驱动力模型是"横向对比"（A行为 vs B行为），心理力学模型就是"纵向对比"（维持当前行为 vs 切换到新行为）。

---

## 核心公式

```
行为改变倾向 = 过程吸引度 + 结果吸引度 
             - 过程排斥度 - 结果排斥度 
             - 当前行为惯性 - 改变成本
```

当行为改变倾向 > 0 时，玩家有可能改变行为；
当行为改变倾向 ≤ 0 时，玩家维持当前行为。

---

## 六因素详解

### 1. 过程吸引度

行为过程中的体验是否愉悦。

| 来源 | 游戏设计中的表现 |
|------|------------------|
| 核心玩法乐趣 | 操作手感、战斗爽感、探索乐趣 |
| 心流状态 | 技能与挑战的匹配程度 |
| 感知流畅度 | 无无效进程、UI响应即时 |
| 审美愉悦 | 视觉、听觉、叙事的沉浸感 |

### 2. 结果吸引度

行为结果是否有吸引力。

| 来源 | 游戏设计中的表现 |
|------|------------------|
| 正反馈强度 | 奖励的数值/稀有度/多样化 |
| 信号预期 | 通过条件反射激活的期待感 |
| 成就感 | 技巧验证、克服挑战的满足 |
| 目标可见性 | 越接近目标追求力度越强（目标梯度效应） |

### 3. 过程排斥度

行为过程中的负面体验。

| 来源 | 游戏设计中的表现 |
|------|------------------|
| 挫败感 | 频繁失败、无法掌握 |
| 无聊 | 难度过低、行为重复无变化 |
| 焦虑 | 唤醒水平过高（耶克斯-多德森定律） |
| 疲劳 | 注意力消耗、操作负担 |
| 无效进程 | 无意义的等待/跑图/菜单操作 |

### 4. 结果排斥度

行为结果的负面效应。

| 来源 | 游戏设计中的表现 |
|------|------------------|
| 负反馈强度 | 惩罚的损失感知（损失厌恶：约为收益的2倍） |
| 沉没成本感知 | 投入时间/资源后"不值得"的感受 |
| 负面预期 | 玩家预期结果不值得努力 |

### 5. 当前行为惯性

维持当前行为的阻力。这是心理力学模型区别于驱动力模型的核心因素——即使另一个行为的有更高吸引力，"惯性"也会阻止玩家切换。

| 来源 | 游戏设计中的表现 |
|------|------------------|
| 路径依赖 | 已有Build/构筑不愿更换（协同效应过强导致） |
| 熟练度投资 | 已花费大量时间练习当前玩法 |
| 习惯形成 | 玩家形成固定的操作/决策模式 |
| 禀赋效应 | 已拥有的物品/角色被赋予更高价值 |

### 6. 改变成本

切换到新行为的门槛。

| 来源 | 游戏设计中的表现 |
|------|------------------|
| 学习成本 | 新机制/新系统的理解门槛 |
| 资源投入 | 切换Build需要的资源成本 |
| 机会成本 | 切换后可能错过的当前行为收益 |
| 心理门槛 | "迈出第一步"的阻力 |

## 与驱动力模型的关系

心理力学模型与驱动力模型（`21-驱动力模型.md`）是互补关系：

```
驱动力模型：A行为 vs B行为 → 选择驱动力更高的
心理力学：维持当前行为 vs 切换到新行为 → 改变倾向 > 0 时才切换

实际决策过程：
1. 先用心理力学：判断是否值得切换行为
2. 再用驱动力模型：如果切换，选择哪个新行为
```

**示例**——自走棋中的阵容切换：

```
当前阵容：6骑士（已投入大量金币升星）
备选阵容：4龙（虽强但需要重新收集）

心理力学分析：
  过程吸引度：新阵容凑齐后有新鲜感 (+) 
  结果吸引度：新阵容强度更高 (+)
  过程排斥度：凑新阵容期间可能掉血 (-)
  结果排斥度：若切换失败损失更大 (-)
  当前行为惯性：已有2个3星骑士，舍不得换 (--)
  改变成本：需要卖掉现有棋子，亏利息 (-)

结果：改变倾向 < 0 → 玩家继续维持当前阵容（路径依赖）
```

## 与正反馈/负反馈循环的关系

### 正反馈循环如何影响心理力学

```
正反馈 → 增加过程吸引度（行为过程中获得乐趣）
       → 增加结果吸引度（奖励预期升级）
       → 增加当前行为惯性（路径依赖加深）
       
结果：正反馈既有利于维持当前行为，也增加了切换的难度
```

### 负反馈循环如何影响心理力学

```
负反馈 → 增加过程排斥度（挫败感、焦虑）
       → 增加结果排斥度（惩罚感知）
       → 减少当前行为惯性（为切换创造条件）

结果：适度的负反馈可以打破路径依赖，引导玩家尝试新行为
```

## 与信号体系的关系

信号（巴甫洛夫条件反射）主要通过影响**过程吸引度**和**结果吸引度**来发挥作用：

```
信号 → 激活期待感 → 增加过程吸引度（"期待的过程本身就是享受"）
                    增加结果吸引度（信号暗示即将到来的奖励）
```

---

## 与强化程式的关系

不同强化程式对心理力学各因素的影响：

| 程式 | 过程吸引度 | 结果吸引度 | 行为惯性 |
|------|-----------|-----------|---------|
| 连续强化 | 每次都有正反馈，过程愉悦 | 稳定但可预测，吸引力低 | 建立快但消退快 |
| 固定比率 | 渐近奖励时有期待 | 可预期，中等 | 稳定，强化后短暂下降 |
| **可变比率** | 不确定产生持续期待 | 惊喜峰值，吸引力高 | **最强最持久** |
| 固定时距 | 临近时上升 | 刚获时最低 | 周期性波动 |
| 可变时距 | 持续轻度期待 | 随机，中等 | 稳定偏持久 |

## 与SDT自我决定理论的关系

SDT三大需求（`48-自我决定理论SDT.md`）对应心理力学中的核心因素：

| SDT需求 | 对应心理力学因素 | 设计策略 |
|---------|-----------------|----------|
| **胜任感** | 过程吸引度 - 过程排斥度 | 平衡难度，确保"我能做到"的体验 |
| **自主感** | 过程吸引度 - 改变成本 | 权力下放降低改变成本 |
| **归属感** | 过程吸引度 + 结果吸引度 | 社交反馈增加过程/结果吸引力 |

## 设计应用

### 1. 引导行为切换

当需要引导玩家从行为A切换到行为B时：

```
策略1：降低当前行为惯性
  - 引入"衰退机制"（装备耐久、消耗品）
  - 加入"刷新/重置"机会（新赛季、重置）
  - 限制持续收益（递减曲线）

策略2：降低改变成本
  - 权力下放（玩家自主选择切换时机）
  - 降低学习成本（渐进式教程）
  - 提供"试水"机会（免费试用、临时体验）

策略3：提高新行为吸引力
  - 信号包装（不揭示奖励全貌，激发期待）
  - 目标梯度效应（让新行为的近期收益可见）
  - 双轨收益（固定保底 + 随机惊喜）
```

### 2. 维持行为（防止过早切换）

当希望玩家持续当前行为时：

```
策略1：增加结果吸引度
  - 可变比率强化（维持期待感）
  - 峰终定律应用（确保每一次结束在正向体验）
  - 协同效应（1+1>2的乘数回报）

策略2：降低过程排斥度
  - 减少无效进程
  - 节奏调节（耶克斯-多德森）
  - 权力下放（玩家自主调节节奏）

策略3：增加改变成本（⚠️ 慎用）
  - 适度沉没成本（不破坏内驱的前提下）
  - 避免让切换成本过高导致"锁死"
```

### 3. 审查要点

| 问题 | 目的 |
|------|------|
| 玩家当前行为的"惯性"来自哪里？ | 识别路径依赖来源 |
| 切换成本是否过高导致玩家被"锁死"？ | 避免负面体验 |
| 新行为的过程吸引度是否足够？ | 确保玩家愿意尝试 |
| 是否有机制帮助玩家克服"迈出第一步"的心理门槛？ | 降低改变成本 |
| 正反馈是否同时增加了"惯性"？是否需要配合调节机制？ | 平衡正反馈与路径依赖 |
