# 学习与遗忘曲线

> 标签：`player-psychology`, `design-philosophy`
> 来源：GameDesignV0.1 补充概念

## 学习曲线

玩家学习游戏机制的过程曲线。好的学习曲线应满足：
- **初始段平坦**：快速上手，建立胜任感
- **中段有梯度**：逐步增加复杂度
- **持续有新鲜感**：新机制在游戏进程中持续引入

## 遗忘曲线（艾宾浩斯）

玩家会遗忘学过的内容。设计上应考虑：
- 关键机制应在多个场景中重复强化
- 长期未使用的机制应有"复习"机会
- 复杂规则应有文档/教程可查阅

## 游戏设计应用

| 阶段   | 策略                         |
| ------ | ---------------------------- |
| 上手期 | 引导式教学，避免信息过载     |
| 成长期 | 逐步解锁新机制，保持学习节奏 |
| 精通期 | 深度系统让玩家持续探索       |

## 塑造（Shaping）——补充

> 完整展开见：`游戏设计理论/02-玩家心理学.md §7.2`

塑造是指通过**逐步强化越来越接近目标的行为**，最终让个体学会一个复杂行为。训练者不等到完美行为出现才给奖励，而是对每一步"近似"都给予强化。

### 与学习曲线的配合

| 学习阶段           | 塑造策略             | 强化方式                     |
| :----------------- | :------------------- | :--------------------------- |
| **初始段（平坦）** | 强化最简单的近似行为 | 连续强化——每次正确都奖励     |
| **中段（梯度）**   | 逐步提高"近似"的标准 | 转为固定比率——多次正确才奖励 |
| **精通期（深度）** | 只强化最精确的行为   | 可变比率——维持行为频率       |

### 与渐进式教程的关系

渐进式教程（先教基础→再教进阶→再教组合）本质上是塑造的学习应用。每个教学步骤都：
1. 定义明确的"近似"目标
2. 给予即时正反馈
3. 逐步提高标准

这也是 CCST 四步关卡设计法（安全呈现→加入危险→更大挑战→机制结合）的心理学基础。

## 关联

- 学习曲线中的"平坦段"对应心流通道
- 遗忘曲线提示关键机制需要重复暴露
- 塑造(Shaping)为渐进式教程提供了心理学基础
- 可参考"玩具系统"的概念——有深度的系统让学无止境
