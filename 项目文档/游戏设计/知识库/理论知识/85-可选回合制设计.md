# 可选回合制设计

> 工作室核心设计方向之一。
> 来源：千水《游戏创作心灵鸡汤》#4 + 变量/电竞教父项目实践
> 关联概念：`36-心流通道.md`, `20-难度调节方案.md`, `17-快车道设计.md`

---

## 一、定义

### 1.1 核心：心流曲线思维，一种系统性的约束

**可选回合制（Optional Turn System）** 的核心不是"跳过回合"这个操作，也不是"让玩家自己控制强度增长"——而是**一种通过系统性约束来调节玩家强度增长的设计思维方式**。

这套设计的推导链是：

```
第一步：观察到问题
  协同效应 → 玩家强度曲线天然不平滑 → 跳脱心流通道
  （不是"设计缺陷"，而是协同游戏的天然特征）

第二步：意识到传统解法的问题
  DDA（系统暗箱调难度）→ 被玩家发现 → 归因到系统 → 破坏严肃性
  硬性数值封顶 → 玩家觉得"被针对"

第三步：找到解法——系统性约束
  不调难度，不封数值，而是设计一套成本和取舍规则——
  让「强度增长」本身附带一个"自然代价"。
  这个代价使得玩家在追求最优解时，自然而然地把心流曲线拉回通道内。
```

**关键区别**：玩家并不是在"上位视角调控自己的强度"。玩家只是在玩自己的游戏——做最优决策、追求收益最大化。而**系统设计的成本和取舍规则，使得这个最优决策路径恰好与心流通道重合**。

```
玩家的视角：  这回合要不要抽塔？——想抽就抽，不想抽就不抽
实际发生的事：玩家过强→不需要装备→不抽塔→运气积累→强度不增长
心流效果：    玩家强度自然停滞，难度追上来 → 自动回到心流通道

玩家从头到尾没有想过"我要调节心流曲线"——他只是做出了"我不需要装备"的决策，
系统规则让这个决策附带调节了强度增长节奏。
```

### 1.2 与DDA（动态难度调节）的本质区别

|        维度        | 传统DDA                | 可选回合制（系统性约束）                   |
| :----------------: | ---------------------- | ------------------------------------------ |
|     **调节者**     | 系统暗箱调节           | **没人调节——规则本身在约束**               |
|    **玩家感知**    | "系统在搞我/帮我"      | "这是我自己的决策结果"                     |
| **被发现后的后果** | 破坏严肃性，玩家反感   | 不存在"被发现"——规则就是规则，从头到尾透明 |
|    **归因方向**    | 外部归因（"系统作弊"） | 内部归因（"我选择冒这个险"）               |
|    **失败模式**    | 橡皮筋效应             | 无（玩家不会怀疑系统在搞自己）             |

### 1.3 一句话总结

> **可选回合制不是让玩家"主动调节节奏"，而是设计一套规则，让玩家在追求最优解时，自动回到心流通道内。**

## 二、理论根基

### 2.1 心流通道理论（核心根基）

可选回合制的思考起点是心流曲线。它不是"给玩家一个跳过按钮"，也不是"让玩家主动调节节奏"——而是：

|            问题            | 传统回答              | 可选回合制的回答                                                         |
| :------------------------: | --------------------- | ------------------------------------------------------------------------ |
| **强度曲线不平滑怎么办？** | 系统动态调难度（DDA） | **设计规则，让玩家追求最优时，强度自然不平滑**                           |
|  **玩家过强无聊怎么办？**  | 提高敌人数值          | **玩家过强→不需要继续变强→系统规则让不增长更有益→停在原地等难度追上来**  |
|  **玩家过弱焦虑怎么办？**  | 降低敌人数值/给buff   | **玩家需要变强→系统规则让增长可得但有代价→自己决定要不要承担代价去变强** |

**关键洞察**：玩家不是在下意识地"调节节奏"。玩家只是在做**局部的最优决策**。是系统规则把这些局部最优决策累加起来，形成了整体的心流调节效果。

### 2.2 利益一致（玩家动机与心流效果的同向性）

这不是"权力下放"（给玩家选择权），而是**让玩家的理性选择和心流状态在方向上保持一致**：

```
玩家动机（局部）：    我要变强 / 我需要装备 / 我不想冒险
       ↓
系统规则（全局）：    变强有代价 / 不抽塔攒运气 / 高费高风险
       ↓
实际效果（心流）：    玩家自然在过强时慢下来，过弱时冲上去
```

这个对齐链条中，玩家从来没有"心流"这个概念——他只是在算自己的小账。系统规则确保他算小账的结果，恰好对心流有利。

### 2.3 归因修正

当系统机制自然约束玩家强度时，翻车了：

- "是我太贪了，不该这时候抽塔的" → 🧠 归因到自己
- "这Boss数值不合理" → ❌ 归因到系统

注意，这也不是"因为玩家有选择权所以归因到自身"——玩家翻车的直接原因看起来就是自己的决策，和选择权没有关系。

### 2.4 快车道（Quick Lane）的关联

快车道和可选回合制是**可以结合也可以独立存在**的两种设计。

它们的共通点：都意识到"玩家过强时会无聊"这个心流问题。

它们的区别：

```
快车道：玩家过强→跳过无聊的挑战→快速推进到更有挑战性的内容
         核心动作是"跳过内容"

可选回合制：玩家过强→系统规则让不增长更有益→自然等难度追上来
           核心动作是"约束增长"
```

两者可以**结合**使用（如《变量》——既有跳关机制，又有不抽塔的运气积累），也可以**独立**使用（如《电竞教父》只有能量系统，没有快车道跳关）。

## 三、两种实现形态与设计模式

可选回合制目前有两个已知的实现形态。它们不是"进化关系"，而是**在同一设计思路下的两种独立实现**——解决同一问题（心流曲线不平滑）的不同方式。

### 形态A：放弃增长型

**代表**：《变量》的"不抽塔攒运气"

#### 具体机制

每回合提供"抽塔/不抽塔"的二元选择：

```
不抽塔 → 运气值增长 → 未来装备概率提高
连续多回合不抽塔 → 运气额外增长
```

#### 约束逻辑

不抽塔=不获得装备=强度不增长。但"不增长"本身是有好处的（攒运气），所以当玩家已经够强时，不抽塔成为合理选择。

#### 玩家视角

"我现在够强了，不需要更多装备，攒着以后用。"

#### 通用模式

```
机制：每回合提供"行动/放弃"的二元选择，放弃时积累未来收益
约束逻辑：不行动=不增长，且不行动积累的好处让"放弃"成为合理选择
代价形式：放弃当次收益机会
调节粒度：粗（二元）
适用场景：轻度策略/塔防
复杂度：⭐
```

---

### 形态B：能量消耗型

**代表**：《电竞教父》的8费系统

#### 具体机制

```
从第10回合起：
  每回合 +1 能量（上限8）
  能量到达8 → 可以行动
  行动时：自由选择打 N 张卡牌（消耗对应能量）
          → 消耗越多 = 场上强度提升越多 = 下次行动来得越晚
  无合适操作 → 可直接过回合（跳过）
  回合硬上限：10回合
```

#### 约束逻辑

能量有限，有"到8才能动"的门槛。能量花得越多=下次来得越晚。玩家自然在优势时少花（够用就行），劣势时多花（搏一把）。

#### 玩家视角

"这回合用多少费？——够用就行，没必要全花光。"

#### 通用模式

```
机制：能量有上限，行动消耗能量，消耗越多下次行动越晚（有行动门槛）
约束逻辑：能量花完=行动权丢掉，需要等能量恢复到门槛才能再行动
代价形式：等待更久才能再次行动
调节粒度：细（0~8连续谱）
适用场景：深度策略卡牌
复杂度：⭐⭐⭐
```

> 两种模式的核心相同：**都设计了一条"变强必有代价"的规则，玩家追求最优解时，这个代价会自然约束强度增长节奏**。区别在于"代价"的形式——放弃增长型的代价=放弃当次收益，能量消耗型的代价=等待更久才能动。

---

### 两种形态的对比

|        维度        |           放弃增长型（变量）           |       能量消耗型（电竞教父）       |
| :----------------: | :------------------------------------: | :--------------------------------: |
|      约束方式      |          不行动=不增长+攒运气          | 行动消耗能量=下次更晚，有行动门槛  |
|      玩家决策      |             1维（抽/不抽）             |     3维（打几张/哪张/过不过）      |
|      调节粒度      |               粗（二元）               |          细（0~8连续谱）           |
| 不增长期的玩家活动 | **继续推进游戏**，运气值作为副产物增长 | **等待能量恢复**，同时观察对手行动 |
|      适用场景      |             轻度策略/塔防              |            深度策略卡牌            |

---

### 与可选回合制关联但不等同的模式

以下模式与可选回合制有表面相似性，但**不包含心流曲线调节的意图**，不应混淆：

|                    模式                     |             相似点              | 本质区别                                                                                                     |
| :-----------------------------------------: | :-----------------------------: | ------------------------------------------------------------------------------------------------------------ |
| **时间代价型**（荒野与曙光：费用=时间推进） | 行动都有"时间代价"，高费=高风险 | 目的是创造"高风险高回报"的决策，不是调节强度增长节奏。玩家打高费不是因为"不想增长太快"，而是"值不值这个风险" |
|         **冷却/CD型**（The Bazaar）         |        卡牌使用后需冷却         | 冷却时间是固定的，玩家无法调节。约束的是单卡使用频率，不是整体强度增长节奏                                   |

## 四、设计要点

### 4.1 约束必须自然，不能"为了调节而调节"

如果玩家能感觉到"这个机制是为了限制我强度而存在的"，那就失败了。

```
✅ 好约束：抽塔的运气积累——玩家觉得"攒着以后用"，不觉得"系统不让我抽"
✅ 好约束：能量8费——玩家觉得"没到8不能动"，不觉得"系统限我"
❌ 坏约束：每隔X回合强制休息——玩家觉得"为什么不能让我继续玩"
```

**检验标准**：玩家能在不知道"可选回合制"这个设计概念的情况下，自然接受这条规则吗？

### 4.2 增长代价决定了调节精度

系统性约束的调节精度取决于"增长的代价"有多精细：

```
粗粒度：跳/不跳（二元）→ 调节精度低，但简单直接
细粒度：0~8费连续谱 → 调节精度高，玩家可微调
```

### 4.3 "不增长"不能是空窗期

当玩家选择不增长时，这段时间不能毫无内容：

|    模式    | 等待期玩家在做什么                |
| :--------: | --------------------------------- |
| 跳过积累型 | 看着运气值增长（有进度反馈）      |
| 能量消耗型 | 观察对手行动/棋盘变化（获取信息） |

### 4.4 不要有全局最优解

可选回合制的一个隐患是：如果经过计算发现"永远不抽塔""永远打最低费"是最优策略，那这个机制就退化为单一最优解，失去了所有策略深度。

**解法**：让每个选择都有适用的场景——有时候需要抽塔（急需装备），有时候不需要（已经够了）；有时候该打高费（急需翻盘），有时候不该（稳扎稳打）。

### 4.5 玩家需要知道具体规则，不需要知道设计概念

玩家必须**被明确告知具体规则**（"不抽塔会积累运气""能量到8才能行动"），否则规则无法引导行为。

但玩家**不需要知道"可选回合制"这个设计概念的存在**——他们不需要知道"设计师通过这个机制在调节我的心流曲线"。他们只需要知道"不抽塔攒运气"就够了。

**检验标准**：玩家理解的规则=实际规则（透明），但玩家的归因是"我在做自己的决策"而非"系统在调节我的节奏"。

### 4.6 多种费用档位支撑决策空间

能量系统的可选回合制需要多种费用档位的卡牌供选择，否则"自由选择打N张"的调节空间是假的。低费垫步调、中费执行战术、高费拼爆发。

## 五、适用判断

| 适合使用可选回合制（系统性约束思维）     | 不适合使用                               |
| ---------------------------------------- | ---------------------------------------- |
| 有协同效应的构筑游戏——强度曲线天然不平滑 | 纯解谜/推理游戏——玩家强度不变，只有对/错 |
| "变强"有明确的代价形式（时间/能量/运气） | "变强"无代价或代价不可感知               |
| 玩家可以基于当前局势做出"值不值得"的判断 | 玩家无法评估当前局势（信息不足）         |
| 母系设计——允许不同玩家有不同节奏         | 父系设计——设计师需要严格控制情感弧线     |

## 六、与知识库其他概念的关系

| 概念           | 关系                                                                                                                            |
| :------------- | :------------------------------------------------------------------------------------------------------------------------------ |
| 心流通道       | **可选回合制的思考起点**——心流曲线不平滑是问题的根源                                                                            |
| DDA            | 要解决的问题相同（强度匹配），但路径相反——DDA调难度，可选回合制约束增长                                                         |
| 快车道         | 可以结合使用，但不是包含关系。快车道跳过内容，可选回合制约束增长                                                                |
| 费用即时间代价 | 可选回合制的能量消耗型与"费用=时间代价"有表面相似性（都有时间成本），但本质区别：可选回合制调节心流曲线，时间代价型调节风险回报 |
| 无效进程       | "等待期"不能变成空窗期                                                                                                          |
| 归因理论       | 归因到自身是可选回合制的副产物，不是设计目标                                                                                    |
