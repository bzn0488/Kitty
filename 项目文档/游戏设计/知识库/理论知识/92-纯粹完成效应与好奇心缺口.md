# 纯粹完成效应与好奇心缺口

> 标签：`player-psychology`, `reinforcement`
> 关联概念：`46-正反馈与负反馈循环.md`, `22-权力下放设计.md`
> 来源：Zeigarnik（1927）, Loewenstein（1994）

## 纯粹完成效应

### 定义
完成一件事本身即是奖励，不依赖完成后的外部奖励。

### 游戏设计中的应用
- **图鉴收集**：全收集的满足感不依赖收集奖励，完成本身即是目标
- **成就系统**：成就点数没有实际用途，但玩家仍然追求
- **100%进度**：地图全开、任务全清、全成就达成
- **对称/序列完成**：收集一套物品（1/6 到 6/6）每获得一个都接近完成

### 正反馈产生机制
未完成 → 心理张力 → 逐步完成 → 张力释放 → 满足感

### 与双轨收益的区别
- 双轨收益依赖外部奖励（固定+随机）
- 纯粹完成效应依赖**内部动机**（"想做完"本身的欲望）

## 好奇心缺口

### 定义
信息不完整时产生填补的冲动（Loewenstein 1994）。

### 游戏设计中的应用
- **未探索地图**：战争迷雾驱动探索
- **未读剧情**：神秘事件驱动推进
- **未解之谜**：隐藏任务、彩蛋、谜题
- **预告/吊胃口**：赛季预告、新英雄剪影

### 好奇心缺口 vs 蔡格尼克效应

| 维度 | 蔡格尼克效应 | 好奇心缺口 |
|------|-------------|-----------|
| 触发 | 任务被中断（做了但没做完） | 信息不完整（知道存在但没见过） |
| 驱动力 | 完成任务释放张力 | 填补信息空白 |
| 应用 | 支线任务、每日任务 | 战争迷雾、谜题、剧情悬念 |
| 强度 | 中等，随时间衰减 | 较强，与信息重要性正相关 |

## 组合应用

```
收集系统 = 纯粹完成效应（全收集欲望）+ 好奇心缺口（还差什么？）
          + 禀赋效应（已有的不想丢） + 目标梯度效应（越接近越努力）
```

四个心理学机制叠加形成最强的收集驱动力。
