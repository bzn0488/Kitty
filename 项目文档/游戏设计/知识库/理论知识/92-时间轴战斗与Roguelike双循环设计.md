# 时间轴战斗系统设计（完整整合）

> 综合调研文档。整合自三篇独立调研（92-双循环框架、93-费用平衡、94-怪物设计），
> 针对荒野与曙光的"时间轴驱动回合制卡牌战斗×Roguelike探索"复合类型，
> 覆盖时间轴费用系统、拼刀博弈、双循环设计、怪物/Boss设计、部位破坏、构筑适配全部内容。

---

## 第一部分：时间轴战斗基础

## 一、时间轴系统的设计模型

### 1.1 时间轴系统的三种模型

| 模型           | 描述                                | 代表游戏           | 适用性                   |
| -------------- | ----------------------------------- | ------------------ | ------------------------ |
| **速度排序型** | 按速度属性排行动顺序，每回合重置    | 火焰纹章、最终幻想 | ❌ 不适用于卡牌主导的战斗 |
| **进度条型**   | 每个人物有独立进度条，填满即行动    | 八方旅人、FF ATB   | ⚠️ 部分适用               |
| **费用时间型** | 行动消耗时间/费用，影响下次行动时机 | 杀戮尖塔、欺诈之地 | ✅ 最接近荒野设计         |

**荒野的独特创新**：费用=时间推进——0费不推进，N费推进N回合。

### 1.2 操作窗口设计

| 模式         | 操作时机                 | 代表游戏   | 体验效果             |
| ------------ | ------------------------ | ---------- | -------------------- |
| **回合制**   | 己方回合集中操作         | 杀戮尖塔   | 策略性强，节奏清晰   |
| **实时暂停** | 随时暂停安排指令         | 博德之门3  | 灵活但决策压力大     |
| **动态窗口** | 行动条到达特定位置时操作 | 荒野与曙光 | 介于回合制和实时之间 |

### 1.3 意图系统设计模式

| 模式         | 描述                       | 案例     |
| ------------ | -------------------------- | -------- |
| **完全可见** | 玩家看到怪物下回合全部意图 | 杀戮尖塔 |
| **部分可见** | 部分意图隐藏或模糊         | 暗黑地牢 |
| **动态调整** | 意图根据玩家行动变化       | 欺诈之地 |
| **多层意图** | 当前意图+后续意图预览      | FTL      |

**核心原则**：意图系统的关键不是"是否显示"，而是"玩家如何应对"。如果应对手段有限，意图系统就变成"看到→机械应对"的循环。

## 第二部分：费用与回合经济

## 二、"费用=时间推进"的平衡系统

### 2.1 与传统卡牌游戏的费用模型对比

| 维度     | 杀戮尖塔（能量）  | 炉石传说（水晶） | 荒野与曙光（时间）         |
| -------- | ----------------- | ---------------- | -------------------------- |
| 费用本质 | 每回合固定回复3点 | 每回合+1，上限10 | **费用=推进的回合数**      |
| 费用回复 | 每回合开始全回复  | 每回合+1         | **不回复，推进即消耗**     |
| 0费卡    | 免费，白赚节奏    | 免费，价值保守   | **不推进时间，但占抽牌位** |
| 高费卡   | 3费强力但需攒能量 | 7费以上后期才有  | **N费=怪物获得N回合行动**  |

### 2.2 费用价值公式

```
牌面价值 = 效果强度 + 时间成本
时间成本 = 费用 × 平均每回合怪物伤害预期

实例（怪物预期伤害=5）：
- 0费卡：时间成本=0，效果>0即可
- 1费卡：时间成本=5，效果需>5（≈基础攻击卡）
- 2费卡：时间成本=10，效果需>10（≈中等强度卡）
- 3费卡：时间成本=15，效果需>15（≈终结技，需附加时间补偿）
```

### 2.3 三种时间成本补偿机制

| 机制         | 原理                           | 适用性           |
| ------------ | ------------------------------ | ---------------- |
| **行动加速** | 高费卡附带减速时间轴效果       | ✅ 可直接应用     |
| **价值前置** | 高费卡在打出回合就产生主要收益 | ✅ 核心设计原则   |
| **时间折扣** | 高费卡减少后续等待时间         | ⚠️ 需避免无限连锁 |

### 2.4 减费机制的特殊风险

时间轴战斗中的减费比传统卡牌游戏更危险——**减费不仅让卡牌更强，还让时间成本降低**，形成双重收益。

**安全设计**：
- 减费有下限（不能减到0以下）
- 减费是暂时的（一回合）而非永久的
- 减费附带条件（如"本回合每打出2张卡，下一张费用-1"）

**危险设计**：
- 永久减费 → 0费高价值卡循环
- 减费叠加 → 费用失控
- 0费抽牌+0费伤害 → 无限连锁

## 第三部分：拼刀与部位破坏

## 三、拼刀系统的博弈深度设计

### 3.1 拼刀的经济学本质

拼刀（进攻即防守）本质上是将**攻击和防御合并为一个操作**。策略从"是否防御"转移到了"攻击哪个部位"和"用多强的攻击"。

**扩展参考**：

| 游戏         | 拼刀/格挡机制         | 可借鉴点                           |
| ------------ | --------------------- | ---------------------------------- |
| **只狼**     | 架势条+完美格挡       | 拼刀是积累优势的过程，而非简单抵消 |
| **死亡细胞** | 招架=无敌帧+反伤      | 高风险高回报的即时反应             |
| **荣耀战魂** | 方向格挡+破防三角     | 攻防博弈的石头剪刀布循环           |
| **欺诈之地** | 攻防双数值+行动点分配 | 同一资源在攻防之间的分配决策       |

### 3.2 拼刀的三种深度维度

| 维度         | 描述                           | 设计实现                     |
| ------------ | ------------------------------ | ---------------------------- |
| **部位选择** | 不同部位对应不同效果           | 类似只狼躯干/血量双系统      |
| **时机选择** | 在时间轴上的哪个节点出手       | 蓄力/快速攻击的差异化时机    |
| **资源分配** | 输出分配到多个部位还是集中一个 | 部位破坏收益 vs 集中击杀收益 |

### 3.3 部位对阵的博弈矩阵

```
          怪物意图
         部位A(攻击)  部位B(防御)  部位C(辅助)
玩家 ┌─────────────────────────────────────
攻击A│  拼刀成功    受到B防御    受到C辅助
攻击B│  受到A攻击   拼刀成功     部分抵消
攻击C│  受到A攻击   受到B防御    拼刀成功
```

**要求**：
1. 没有绝对安全的选项（每个选择都有风险）
2. 信息不对称（玩家可根据意图选针对）
3. 收益差异化（成功拼刀攻击部位 > 防御部位 > 失败）

**避免"看见→匹配"机械操作的方案**：
- **部分随机**：怪物意图可见但有随机偏移量
- **多层意图**：看到"即将攻击"但不知具体部位
- **动态切换**：怪物在玩家行动后调整意图（欺诈之地模式）

## 四、部位破坏的深度设计

### 4.1 部位破坏的经济学

部位破坏是**玩家用输出资源换取战术优势**的决策，而非额外奖励：

```
方案A：集中攻一个部位 → 破坏后获战术优势
方案B：分散攻多个部位 → 多个部分损伤
方案C：集中攻本体 → 加快战斗进程

关键：三种方案都应可行，取决于构筑和战局。
```

### 4.2 部位类型与效果

| 部位类型                   | 破坏前      | 破坏后         | 破坏难度       |
| -------------------------- | ----------- | -------------- | -------------- |
| **武器部位**（爪/牙/炮管） | 高攻击力    | 攻击力大幅下降 | 中             |
| **防御部位**（甲壳/盾牌）  | 高防御/减伤 | 伤害提升       | 高（需先破防） |
| **机动部位**（翼/足）      | 高速/多动   | 减速           | 低             |
| **特殊部位**（眼/核心）    | 特殊能力    | 失去能力       | 中高           |
| **弱点部位**               | 受伤害加倍  | 持续暴露       | 条件触发       |

### 4.3 部位恢复与正反馈控制

部位破坏永久化会导致战斗越简单（正反馈）。控制手段：

- **部位重生**：部分部位N回合后再生（周期性挑战）
- **狂暴补偿**：Boss失去部位后获得强化（愤怒机制）
- **阶段重置**：新阶段时部位恢复

**平衡基线**：破坏一个部位的投入 ≈ 造成30%本体HP伤害的投入。

### 4.4 时间轴上的破坏节奏

```
玩家打出3费卡（推进3回合）
  ├── 第1回合：攻击部位A → 部分损伤
  ├── 第2回合：怪物行动（可能攻击玩家）
  └── 第3回合：继续攻击 → 部位破坏！
```

## 第四部分：怪物与Boss设计

## 五、时间轴战斗中的怪物设计

### 5.1 怪物时间轴定位模型

| 模型         | 描述                                    | 适用场景         |
| ------------ | --------------------------------------- | ---------------- |
| **同步节点** | 怪物在固定时间节点行动                  | 节奏规整的战斗   |
| **速度驱动** | 怪物有独立速度，按速度填行动条          | 差异化速度怪物   |
| **费用联动** | 怪物行动与玩家费用挂钩（N费=怪物动N次） | **荒野核心模型** |

**费用联动特殊性**：怪物行动频率直接受玩家决策影响。玩家可用低费连打压制怪物行动，但低费卡效果弱，需在"次数优势"和"单卡强度"间权衡。

### 5.2 怪物行动的成本定价

```
怪物每次行动的价值 ≈ 1费卡的平均价值

意味着：
- 1费卡的效果 ≈ 抵消1次怪物行动 + 微弱收益
- 2费卡的效果需要 > 2次怪物行动总损失 + 可观收益
- 怪物基线：每单位时间造成约1费卡价值的压力
```

### 5.3 怪物行为设计模式目录

| 模式          | 描述                      | 玩家应对                | 难度 |
| ------------- | ------------------------- | ----------------------- | ---- |
| **蓄力重击**  | 蓄力1-2回合后释放高伤害   | 打断/减伤/加速击杀      | 初级 |
| **弹幕射击**  | 多发低伤害分散到多个部位  | 拼刀覆盖面 vs 集中格挡  | 初级 |
| **召唤援军**  | 召唤杂兵辅助              | AOE清理 vs 速杀主体     | 中级 |
| **狂暴模式**  | HP低于阈值后攻速/伤害提升 | 控制节奏提前爆发        | 中级 |
| **反伤/荆棘** | 攻击怪物时自伤            | 调整输出节奏/用持续效果 | 中级 |
| **复制/镜像** | 复制玩家的卡牌或行为      | 谨慎选择使用哪些卡牌    | 高级 |
| **环境改变**  | 每回合改变战场规则        | 构筑预留应对方案        | 高级 |
| **时间操纵**  | 加速/减速时间轴           | 适应新时间节奏          | 高级 |

### 5.4 意图的三层复杂度

```
第一层：固定循环 → 适合普通怪物
第二层：条件触发 → 适合精英怪（玩家行为影响怪物）
第三层：模式切换（HP阶段） → 适合Boss
```

### 5.5 信号设计（Tell）

- ✅ 蓄力动画、意图图标变化、时间轴标记
- ❌ 无预兆秒杀、信号太短（<1回合）、信号不明显

**时间轴战斗的优势**：可在时间轴上直接标注未来事件（"3回合后将全屏攻击"）。

## 六、Boss战设计框架

### 6.1 Boss战的三段式结构

```
第一阶段：试探期（100%-70% HP）
  ├── Boss展示基础行为模式
  ├── 玩家学习应对
  └── 少量部位暴露

第二阶段：变化期（70%-30% HP）
  ├── Boss解锁新技能
  ├── 关键部位出现
  ├── 战斗节奏加快
  └── 需调整策略

第三阶段：决战期（30%-0% HP）
  ├── Boss进入狂暴/最终形态
  ├── 所有部位暴露（高风险高回报）
  ├── 时间压力最大
  └── 玩家执行完整构筑策略
```

### 6.2 Boss技能设计清单

每个Boss技能应满足：□ 可规避 □ 有代价 □ 有信号（提前1-2回合）□ 有阶段变化 □ 符合主题

### 6.3 Boss类型设计模板

| 类型         | 核心挑战                 | 部位设计            | 参考             |
| ------------ | ------------------------ | ------------------- | ---------------- |
| **巨像型**   | 缓慢但致命，多可破坏部位 | 四肢+核心           | 暗黑地牢猪王     |
| **刺客型**   | 高速移动，频繁改变目标   | 无/少部位，重点闪避 | 杀戮尖塔时光老头 |
| **法师型**   | 改变战场规则，召唤援军   | 法杖/水晶等辅助部位 | 怪物火车契约之王 |
| **机械型**   | 部件协同，破坏顺序是关键 | 多功能部件          | 只狼狮子猿       |
| **巨型Boss** | 在Boss身上移动攻不同位置 | 多个独立区域        | 旺达与巨像       |

## 七、多怪物战斗编排

### 7.1 多怪物时间轴管理

| 模式         | 描述                   | 适用场景   |
| ------------ | ---------------------- | ---------- |
| **同步行动** | 所有怪物同时行动       | 杂兵群     |
| **错位行动** | 不同怪物不同时间节点   | 精英+杂兵  |
| **波次行动** | 第一波消灭后第二波出现 | 长时间战斗 |
| **联动行动** | 怪物A行动触发怪物B行动 | 需策略处理 |

**建议**：最多同时3-4个怪物，超出时玩家难以追踪所有意图。

### 7.2 怪物组合模式

| 模式        | 配置                     | 玩家策略           |
| ----------- | ------------------------ | ------------------ |
| **矛与盾**  | 1高攻+1高防              | 先杀盾还是先杀矛？ |
| **奶与DPS** | 1治疗+1输出              | 优先击杀治疗       |
| **光环组**  | 1光环源+多受增益者       | 击杀光环源         |
| **孵化组**  | 1产卵者+不断生成杂兵     | 控产卵节奏 vs AOE  |
| **互补组**  | 怪物A免疫物理，B免疫法术 | 需双修构筑         |

### 7.3 精英怪的教学功能

精英怪应承担教学功能——教会玩家应对某种挑战以便在Boss战应用：

```
第一幕精英 → 教部位破坏
第二幕精英 → 教应对多怪物
第三幕精英 → 教资源管理
```

**原则**：Boss战不应引入精英战中没出现过的新机制。

## 第五部分：Roguelike双循环与构筑

荒野与曙光的内线（战斗）与外线（探索/领地）双循环是其核心结构：

```
外线循环（慢节奏、战略层）
  ┌──────────────────────────────────────┐
  │  探索选路 → 事件/资源 → 领地建设    │
  │        ↕              ↕              │
  │   遭遇敌人 ←── 进入战斗              │
  └──────────────┬───────────────────────┘
                 │ 触发
                 ▼
  ┌──────────────────────────────────────┐
  │  时间轴推进 → 卡牌操作 → 拼刀结算    │
  │        ↕              ↕              │
  │   怪物意图 → 部位破坏 → 战利品      │
  └──────────────────────────────────────┘
  内线循环（快节奏、战术层）
```

**设计挑战**：两个循环如何相互影响而不是彼此独立？

**解决方案参考**：

| 影响方向          | 机制                                   | 参考游戏              |
| ----------------- | -------------------------------------- | --------------------- |
| **外线→内线**     | 探索中获得的卡牌/装备/资源影响战斗能力 | 杀戮尖塔（遗物+卡牌） |
| **内线→外线**     | 战斗消耗的资源影响探索续航             | 暗黑地牢（压力/疾病） |
| **外线↔内线双向** | 领地建设增强战斗能力，战斗获得领地资源 | 超越光速（FTL）       |

## 八、选路系统设计

### 8.1 选路系统的设计深度

荒野的选路系统需要超越"左中右三选一"的简单模式：

| 模式         | 描述                       | 适用场景                  |
| ------------ | -------------------------- | ------------------------- |
| **分支路径** | 不同路线有不同类型节点分布 | 杀戮尖塔、大多数Roguelike |
| **可见内容** | 玩家看到节点内容后选择     | 怪物火车（战斗前可见）    |
| **探索迷雾** | 部分内容可见，部分隐藏     | 暗黑地牢（房间探索）      |
| **条件路径** | 满足特定条件解锁特殊路径   | 超越光速（FTL跳跃限制）   |

**对荒野的建议**（链接到 `体验4X`）：
- 选路应让玩家有**信息不对称的决策**——知道A方向有战斗但奖励好，B方向安全但收益低
- 选路与领地系统联动——特定路线需要领地解锁或产生领地资源

### 8.2 战斗续航与经济设计

内线战斗消耗的资源需要在外线恢复，这是双循环的核心张力：

```
战斗消耗：
  HP/血量 → 通过事件/休息恢复（外线）
  卡牌消耗 → 通过商店/事件补充（外线）
  金钱/资源 → 通过战斗/探索获得（内外结合）

关键设计问题：每次战斗后玩家状态应恢复到什么程度？
- 完全恢复 → 每次战斗都是独立挑战（杀戮尖塔模式）
- 部分恢复 → 战斗有累积消耗（暗黑地牢模式）
- 不恢复 → 资源管理是核心策略（FTL模式）
```

荒野的定位似乎介于杀戮尖塔和暗黑地牢之间——每次战斗间有部分恢复（通过领地/事件），但战斗中的消耗（HP、卡牌）需要在外线管理。

## 九、Roguelike构筑与成长设计

### 9.1 构筑的三种路线

| 路线         | 核心                        | 适用场景     | 参考游戏           |
| ------------ | --------------------------- | ------------ | ------------------ |
| **构筑驱动** | 卡牌/装备的协同组合决定强度 | 核心战斗系统 | 杀戮尖塔、怪物火车 |
| **数值驱动** | 基础属性成长主导            | 角色养成     | 死亡细胞、哈迪斯   |
| **混合驱动** | 构筑+数值同时成长           | 复合类型     | 荒野与曙光？       |

**荒野的定位**：卡牌构筑（战斗）× 角色成长（养成）× 领地建设（领地）三条成长线并行——这是一个有野心的设计，挑战在于三条线的节奏同步。

### 9.2 增量Roguelike的成长曲线

荒野的增量自走棋基因带来了特殊的成长设计问题：

**典型的Roguelike成长曲线**：
```
强度
  ↑
  |          ┌─── 成型（构筑完成）
  |       ┌──┘
  |    ┌──┘
  | ┌─┘
  |┘
  +──────────────────→ 时间
  初期      中期      后期
```

**增量游戏的成长曲线**：
```
产出
  ↑
  |          ┌─── 幂次增长
  |       ┌──┘
  |    ┌──┘
  | ┌─┘
  |┘
  +──────────────────→ 时间
  每次重置后起点更高
```

**荒野需要解决的问题**：Roguelike的"每次开局重来"与增量游戏的"积累成长"之间的张力。

**可能的融合方案**：
1. **跨局继承**：类似Rogue Legacy——每次失败后部分能力继承到下一局
2. **领地作为存档点**：领地的建设成果不会因战斗失败而消失
3. **分支难度**：玩家可以选择继续当前进度或"重置"换区获取不同资源

### 9.3 背包管理类战斗设计

荒野与曙光的项目经验中包含背包乱斗和背包英雄的分析。进一步补充：

**背包管理的核心设计维度**（链接到 `背包乱斗单位设计`、`背包英雄单位设计`）：

| 维度     | 背包乱斗              | 背包英雄       | 对荒野的参考                   |
| -------- | --------------------- | -------------- | ------------------------------ |
| 空间管理 | 4×4网格，旋转摆放     | 线性背包，堆叠 | 荒野是卡牌手牌，网格不适用     |
| 激活条件 | 相邻物品激活协同      | 物品栏激活技能 | 手牌中的卡牌位置是否影响效果？ |
| 经济循环 | 战斗中掉落的物品+金币 | 商店购买+升级  | 荒野的卡牌获得途径             |

**对荒野的特定参考**：虽然荒野不是背包管理游戏，但背包类的"空间限制产生策略选择"的核心设计原则可迁移到手牌管理——手牌上限就是空间限制，弃牌就是"整理背包"。

## 十、Roguelike中的领地/基地建设

### 10.1 领地系统在Roguelike中的罕见性

荒野是为数不多的在Roguelike中加入领地建设/基地建造的游戏之一。参考案例有限：

| 游戏                    | 基地建设模式         | 与战斗的关系       |
| ----------------------- | -------------------- | ------------------ |
| **《超越光速》（FTL）** | 飞船升级（资源管理） | 直接影响战斗能力   |
| **《暗黑地牢》**        | 城镇建设（全局升级） | 影响英雄招募和装备 |
| **《夜族崛起》**        | 城堡建造（开放世界） | 制造装备和存储资源 |
| **《冬日计划》**        | 营地建设（生存）     | 提供生存增益       |

**核心设计问题**：领地的成长速度如何与战斗难度曲线匹配？
- 领地成长过快 → 战斗太简单
- 领地成长过慢 → 玩家在战斗中感到无力
- 领地与战斗的脱节 → 两个系统各玩各的，失去融合意义

**建议方案**：
- 领地提供的增益应偏向**战术多样性**而非**数值碾压**（解锁新策略而非单纯加数值）
- 领地的建造需要战斗获得的特殊资源
- 战斗中消耗的补给品需要领地生产

### 10.2 蓝图作为卡牌延伸的设计

蓝图的定位是"可带回战斗的领地产出"。这本质上是**跨系统的资源转换**：
```
领地生产 → 蓝图 → 战斗使用
战斗获得 → 蓝图材料 → 领地建设
```

如果蓝图与卡牌系统的接口清晰，这个循环就有闭环。否则容易变成两个独立的游戏。

## 第六部分：难度曲线与自检清单

## 十一、Roguelike难度曲线

### 11.1 怪物难度曲线的两个陷阱

```
陷阱1：线性增长
  怪物强度 = 当前楼层 × 固定系数
  问题：玩家能算出"到X层之后打不过"→失去动力

陷阱2：玩家成长脱节
  怪物强度按预设曲线增长，但玩家构筑可能在某层突然成型
  问题：成型前被打爆，成型后太简单
```

**建议**：采用"自适应难度门槛"——每个区域有基础强度范围，玩家构筑越强则怪物配置越复杂（非数值更高），关键检测点用精英/Boss把关。

## 十二、设计自检清单

### 12.1 费用平衡速查

```
0费卡：效果 > 0 → 抽牌/过牌/微量效果/条件触发
1费卡：效果 > 5 → 标准伤害/防御/单体（基准6-8伤害）
2费卡：效果 > 10 → AOE/强力单体/复合效果（基准12-16伤害）
3费卡：效果 > 15 → 终结技/翻盘/组合核心（需附带时间补偿）
4费+：仅限特定构筑/减费场景
```

### 12.2 怪物自检

- [ ] 至少2种可行的应对策略？
- [ ] 玩家第一次见时1回合内看懂威胁？
- [ ] 行动符合"每次行动 ≈ 1费卡价值"基线？
- [ ] 意图信号清晰可见？
- [ ] 失败原因是"选择错了"而非"不公平"？

### 12.3 Boss战自检

- [ ] 每阶段有独特的挑战重点？
- [ ] 阶段转换有明确信号？
- [ ] 技能符合"可规避+有代价+有信号"？
- [ ] 部位破坏可选而非必须？
- [ ] 战斗长度与阶段匹配？
- [ ] 没有精英战中没出现过的新机制？

### 12.4 部位破坏自检

- [ ] 每个部位破坏后有即时可感知的战斗变化？
- [ ] 不同部位收益有差异？
- [ ] 存在"不破坏部位也能赢"的路线？
- [ ] 投入产出比合理（≈30%本体HP）？
- [ ] 破坏进度清晰可见？

### 12.5 外线-内线闭环自检

- [ ] 每次外线探索后战斗能力有可见变化？
- [ ] 每次战斗后外线资源有可感知增长？
- [ ] 存在"只战斗不探索"的路线？（效率应低但不应封死）
- [ ] 领地建设有阶段性里程碑？
- [ ] 外线决策影响内线打法风格？

### 自问自答

**Q1：这三篇文档的核心贡献是什么？**
合并后的文档提供了荒野与曙光的时间轴战斗系统从底层费用模型到顶层Boss设计的完整设计参考。最独特的价值在于"费用=时间推进"的量化平衡公式——这在现有游戏设计文献中很少见到系统化的处理。

**Q2：内容是否有重复或冲突？**
原三篇文档在拼刀系统和部位破坏上有部分重叠（92§2与93§2、94§4），合并后已去重。在"意图系统"上有不同观点——92强调"完全可见"而94建议"多层隐藏"，合并后保留了两种观点的并存作为可选方案。

**Q3：哪些内容需要调整？**

| 问题                                 | 建议调整                                                                                                     |
| ------------------------------------ | ------------------------------------------------------------------------------------------------------------ |
| **费用公式中的怪物预期伤害是静态的** | 实际游戏中怪物伤害随进程增长，公式应改为动态值 \( \text{时间成本} = \text{费用} \times f(\text{当前阶段}) \) |
| **减费机制分析缺少具体数值边界**     | 需要补充"减费到多少是安全的"具体数值测试建议                                                                 |
| **拼刀博弈矩阵缺少"多角色"场景**     | 荒野可能是单人操作+多角色队伍，拼刀矩阵需扩展                                                                |
| **Boss阶段划分的HP百分比是假设值**   | 需要根据实际战斗节奏调整，不同Boss应有不同的阶段阈值                                                         |
| **操作窗口设计缺少多角色切换场景**   | 时间轴战斗在多角色时如何切换操作、分配资源需要补充                                                           |
