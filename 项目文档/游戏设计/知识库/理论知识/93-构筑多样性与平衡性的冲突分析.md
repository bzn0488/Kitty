# 构筑多样性与平衡性的冲突分析

> 调研目标：深入分析时间轴费用模型对Roguelike构筑多样性的影响，
> 识别不同构筑原型的平衡风险，提供数值模拟基准和设计建议。

---

## 一、时间轴战斗中的构筑原型

### 1.1 四种基本构筑原型

在时间轴费用模型下，卡牌构筑可归纳为四种原型：

| 原型       | 策略                                        | 费用偏好   | 核心卡牌类型        | 优势               | 弱点                             |
| ---------- | ------------------------------------------- | ---------- | ------------------- | ------------------ | -------------------------------- |
| **速攻型** | 堆0-1费卡，每回合多次行动，压制怪物行动次数 | 0-1费      | 抽牌、0费伤害、过牌 | 时间成本低，节奏快 | 单卡强度低，后期乏力             |
| **重击型** | 用高费卡单发高伤害，承担时间风险            | 2-3费      | 高伤害、减费、加速  | 爆发高，一锤定音   | 高时间成本，空白期脆弱           |
| **控制型** | 通过减速/冻结/眩晕控制时间轴                | 各费用混合 | 控制、持续、延迟    | 主动权在握         | 控制效果需足够强才能补偿时间成本 |
| **持续型** | 使用多回合buff/debuff，一劳永逸             | 1-2费      | 持续效果、光环      | 长期收益高         | 价值在时间推进中衰减             |

### 1.2 四种原型在时间轴上的行为差异

```
速攻型的时间轴使用模式：
  玩家：0费→1费→1费→0费→...（几乎不推进时间）
  怪物：极少行动
  问题：单张卡效果弱，依赖卡组循环速度

重击型的时间轴使用模式：
  玩家：蓄力...3费！（推进3回合）→ 等待...→ 3费！
  怪物：多次行动（可能在此期间重创玩家）
  问题：等待期的生存能力

控制型的时间轴使用模式：
  玩家：1费控→1费控→2费输出→...
  怪物：行动被延迟/跳过
  问题：控制效果的价值必须≥被控制的那段"时间"

持续型的时间轴使用模式：
  玩家：2费buff（推进2回合）→ buff持续生效→ 后续出牌享受buff加成
  怪物：在buff期间正常行动
  问题：buff的价值被怪物行动部分抵消
```

## 二、0费速攻型：最危险的平衡风险

### 2.1 0费卡的系统性优势

在时间轴费用模型中，0费卡具有**系统性优势**：

```
0费卡的时间成本 = 0（不推进时间）
1费卡的时间成本 = 怪物1回合的预期伤害

这意味着：即使0费卡的效果只有1费卡的30%，其性价比也高于1费卡。

举例：
  1费卡：造成6点伤害，时间成本5，净收益 = 1
  0费卡：造成2点伤害，时间成本0，净收益 = 2
  → 0费卡的性价比更高！
```

**推论**：如果0费卡的效果 > 0即可，那么任何0费卡（即使是造成1点伤害）都是正收益。玩家会倾向于堆叠0费卡，绕开时间成本机制。

### 2.2 0费卡的11种设计及其平衡

| 类型          | 效果                   | 平衡风险                   | 建议调整                   |
| ------------- | ---------------------- | -------------------------- | -------------------------- |
| 🔴 0费打伤害   | 造成2-3点伤害          | **极高**——纯正收益，无代价 | 改为"条件触发"或"代价触发" |
| 🔴 0费抽牌     | 抽1张牌                | 极高——压缩卡组，加速循环   | 改为"弃一张牌后抽一张"     |
| 🟡 0费过牌     | 抽1弃1                 | 高——过滤卡组               | 可接受，但限制数量         |
| 🟡 0费护甲     | 获得3-5点护甲          | 高——免费防御               | 限制数值或加条件           |
| 🟢 0费debuff   | 给怪物上debuff         | 中——效果需要配合           | 可接受，鼓励配合           |
| 🟢 0费条件触发 | 满足条件时触发额外效果 | 低——不满足条件时是废卡     | 优选设计模式               |
| 🟢 0费连击     | 本回合每打出X张卡触发  | 中——鼓励0费堆叠但有上限    | 设定触发上限               |
| 🟢 0费消耗型   | 使用后从卡组永久移除   | 低——一次性资源             | 安全，适合作为奖励         |
| 🟢 0费切换型   | 改变姿态/模式          | 中——效果间接               | 可接受                     |
| ❓ 0费衍生     | 由其他卡生成的0费卡    | 视效果而定                 | 需单独评估                 |
| ❓ 0费减费后   | 原为1费，被减到0费     | 视原效果而定               | 减费需设下限               |

### 2.3 "纯0费"构筑的可行性评估

假设玩家构筑了一套全部由0费卡组成的卡组：

```
每回合操作：
  打光所有手牌（5-7张0费卡）→ 全部0时间成本
  每张造成2点伤害 → 每回合10-14点伤害
  怪物几乎不行动（因为时间不推进）

问题：
  1. 游戏变成"无脑丢牌"，失去了费用决策的策略深度
  2. 高费卡完全没有出场机会
  3. 所有构筑都趋同于0费
```

**结论**：纯0费构筑必须通过设计手段防止，否则整个费用系统崩溃。

### 2.4 防止0费霸权的设计手段

| 手段                | 原理                                | 副作用             | 推荐程度     |
| ------------------- | ----------------------------------- | ------------------ | ------------ |
| **手牌上限惩罚**    | 每回合结束弃牌到上限，0费卡不能保留 | 对所有构筑都有影响 | ⚠️ 间接但有效 |
| **0费卡数量限制**   | 卡组中0费卡数量有上限               | 限制了构筑自由度   | ✅ 直接有效   |
| **0费卡效果递减**   | 同一回合每打一张0费卡，效果递减     | 增加了复杂度       | 🟡 较优雅     |
| **抽牌费用化**      | 抽牌本身也消耗时间/资源             | 改变基础规则       | ❌ 过于激进   |
| **0费卡有负面效果** | 0费卡附带小额负面效果（如自伤）     | 符合"代价换收益"   | ✅ 推荐       |
| **费用联动收益**    | 使用高费卡获得额外收益（0费卡没有） | 间接激励费用多样化 | ✅ 推荐       |

**推荐组合方案**：
1. 0费卡数量限制（卡组中不超过X张）+ 每回合0费卡使用次数限制
2. 有条件0费（"如果你本回合使用了至少1张2费+卡，本张卡费用-1"）
3. 0费卡的收益应偏向功能性（抽牌/过牌/debuff）而非直接伤害

## 三、高费构筑的可行性问题

### 3.1 高费卡的价值门槛

在时间轴模型中，高费卡面临**双重门槛**：

```
假设怪物预期伤害 = 10（中期）

2费卡：
  时间成本 = 2 × 10 = 20
  卡牌效果需 > 20 才值得用
  作为对比：2张1费卡的成本 = 1×10 + 1×10 = 20（但可以分两回合打出，灵活性更高）

3费卡：
  时间成本 = 3 × 10 = 30
  卡牌效果需 > 30
  同时，玩家需要在这3回合等待期间存活
```

**高费卡的隐性劣势**：
- 灵活性低（不能拆分使用）
- 风险集中（如果被反制/抵消，损失巨大）
- 等待期脆弱（怪物可能在玩家等待时造成致命伤害）

### 3.2 高费卡的必要补偿

为了让高费卡在构筑中有一席之地，以下补偿机制是**必需**的：

| 补偿机制     | 实现方式                               | 量化效果             |
| ------------ | -------------------------------------- | -------------------- |
| **立即生效** | 打出当回合造成全额伤害/效果            | 消除等待期风险       |
| **不可反制** | 不能被怪物的防御/反伤抵消              | 风险溢价             |
| **附带减费** | 使用后下张卡费用-1                     | 间接降低后续时间成本 |
| **超额收益** | 效果超过线性期望（1费=6伤 → 3费>18伤） | 补偿灵活性损失       |
| **全局增益** | 附带永久性buff（即使战斗结束也保留）   | 跨战斗收益           |

### 3.3 费用曲线的设计基线

基于以上分析，推荐的费用分布：

```
卡牌池费用分布（推荐）：
  0费卡：10-15%（主要用于功能性，非伤害）
  1费卡：35-40%（主力费用段）
  2费卡：30-35%（中型效果，核心卡）
  3费卡：10-15%（终结技/翻盘卡，需补偿机制）
  4费+：0-5%（仅在特殊机制下存在）

各费用段的效果强度基准：
  0费：功能性为主，伤害≤3
  1费：伤害6-8 / 等值效果
  2费：伤害12-16（≈2张1费卡之和 × 1.0-1.2）
  3费：伤害20-28（≈3张1费卡之和 × 1.1-1.3）+ 额外补偿
```

## 四、构筑路径的多样性保障

### 4.1 四种构筑原型的强度曲线

```
强度
  ↑
  |  速攻型 ──── 前期最强，后期平坦
  |  ┌───┐
  |  │   └─── 重击型 ──── 前期弱，后期爆发
  |  │       └───┐
  |  │           └─── 控制型 ──── 中期最强
  |  │               └───┐
  |  │                   └─── 持续型 ──── 平稳增长
  |  │
  +──────────────────────────→ 游戏进程
  第一幕    第二幕      第三幕
```

**设计目标**：四种原型的强度曲线应在不同阶段交叉，而不是某一条全程领先。

### 4.2 构筑路径的支撑卡牌

每种构筑原型需要专属的支撑卡牌才能成立：

| 原型       | 必需的核心卡                   | 需要的支撑            | 数量建议   |
| ---------- | ------------------------------ | --------------------- | ---------- |
| **速攻型** | 0费产生器、抽牌引擎、过牌循环  | 至少3张核心+5张支撑   | 占卡组40%+ |
| **重击型** | 高费终结技、减费卡、生存卡     | 2-3张终结技+4-6张支撑 | 占卡组30%  |
| **控制型** | 减速/冻结/眩晕、控场、输出转换 | 4-6张控制+3-4张输出   | 占卡组50%  |
| **持续型** | 强力buff、叠层机制、维持卡     | 2-3张核心buff+保护卡  | 占卡组35%  |

### 4.3 构筑路径的互斥性与协同性

好的构筑设计应让不同路径之间有**部分重叠**（玩家可以在中途转换）但不失去**方向感**：

```
重叠区设计：
  速攻型 ←→ 控制型：共享低费功能性卡
  重击型 ←→ 持续型：共享高费核心卡
  速攻型 ←→ 持续型：少量重叠（过牌机制）
  控制型 ←→ 重击型：少量重叠（减费机制）

完全互斥的设计（应避免）：
  "速攻型专属卡" 只能在速攻构筑中使用
  → 锁死了玩家的转型可能性
```

**更好的设计**：卡牌有"倾向性"但没有"排他性"——速攻卡在重击构筑中效果较弱但仍可使用。

## 五、数值模拟：费用收益的量化分析

### 5.1 简化数学模型

假设：
- 手牌上限 = 7
- 每回合抽牌 = 5
- 怪物预期伤害 = 5（前期）/ 10（中期）/ 15（后期）
- 1费卡基准效果 = 6伤害
- 费用收益比 = 效果 / 费用

```
前期（怪物伤害=5）：
  0费卡：效果2，时间成本0，净收益 = 2
  1费卡：效果6，时间成本5，净收益 = 1
  2费卡：效果13，时间成本10，净收益 = 3
  3费卡：效果22，时间成本15，净收益 = 7

中期（怪物伤害=10）：
  0费卡：效果2，时间成本0，净收益 = 2
  1费卡：效果6，时间成本10，净收益 = -4（亏了！）
  2费卡：效果13，时间成本20，净收益 = -7（亏更多）
  3费卡：效果22，时间成本30，净收益 = -8

除非：
  - 卡牌效果随游戏进程升级（卡牌升级/强化系统）
  - 怪物的伤害预期被玩家的防御/护甲/控制降低
  - 存在独立的"减伤"系统不依赖卡牌效果
```

### 5.2 核心发现

**前期（怪物伤害低）**：
- 高费卡净收益高 → 重击型有优势
- 低费卡净收益低但不亏 → 速攻型可用但不突出

**中期（怪物伤害增长）**：
- 低费卡和高费卡都开始亏损 → **必须升级卡牌或降低怪物伤害**
- 0费卡成为唯一稳定正收益 → **0费霸权风险最高期**

**后期（怪物伤害高）**：
- 所有费用段的净收益都为负 → **必须依靠构筑协同和卡牌升级来弥补**
- 纯数值卡牌失效，只有组合技和协同卡能产出超额价值

### 5.3 对设计的指导意义

1. **卡牌必须能升级或强化**，否则中期以后所有费用段都亏损
2. **0费霸权最危险的时期是中期**——前期伤害低无所谓，后期0费也得升级
3. **怪物伤害的增长必须与卡牌升级同步**，否则玩家会感觉"越来越弱"
4. **协同效果（组合技）的价值需要超过线性叠加**——这是鼓励构筑多样性的核心手段

## 六、构筑多样性的保障策略

### 6.1 七条核心设计原则

| #   | 原则                        | 说明                                       |
| --- | --------------------------- | ------------------------------------------ |
| 1   | **没有绝对最优费用段**      | 每个费用段在不同局势下有不同价值           |
| 2   | **0费卡是功能性而非伤害性** | 0费伤害卡会导致霸权，0费功能性卡鼓励创意   |
| 3   | **高费卡必须附带补偿**      | 没有补偿的高费卡没人用                     |
| 4   | **卡牌需要成长性**          | 随游戏进程升级/强化，否则中期以后亏本      |
| 5   | **协同>堆叠**               | 组合技的价值应超过同类卡简单叠加           |
| 6   | **路径可转换**              | 玩家应能在不同构筑路径间平滑过渡           |
| 7   | **每种原型有专属"爽点"**    | 速攻的爽点是操作密度，重击的爽点是一击必杀 |

### 6.2 构筑路径的"健康检测"清单

- [ ] 0费卡的伤害是否≤3？（避免纯0费伤害构筑）
- [ ] 纯0费卡组是否被机制限制？（上限/递减/条件）
- [ ] 高费卡是否有补偿机制？（立即生效/不可反制/减费）
- [ ] 不同费用段是否都有"必选"的强力卡？
- [ ] 构筑路径之间是否有重叠区？（允许中途转换）
- [ ] 卡牌是否可以升级/强化？（应对怪物伤害增长）
- [ ] 是否存在至少3种可行的最终构筑？（验证多样性）

### 6.3 超出本文范围的相关问题

1. **卡牌升级系统的具体设计**：升级时机、升级方式、升级前后的数值变化
2. **协同组合的发现与引导**：玩家如何发现卡牌间的协同？需要UI引导吗？
3. **随机性对构筑的影响**：商店刷新、卡牌掉落、事件奖励——随机性能否支持定向构筑？
4. **构筑与外部成长的联动**：角色等级/领地建设是否影响构筑选项？
