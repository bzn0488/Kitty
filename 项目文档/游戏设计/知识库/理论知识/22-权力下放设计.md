# 权力下放设计

> 标签：`empowerment`, `player-psychology`
> 关联概念：`48-自我决定理论SDT.md`, `43-游戏节奏设计.md`, `20-难度调节方案.md`

## 定义

开发者主动将原本属于设计者的控制权，有策略地让渡给玩家的设计哲学。

## 四种权力类型

| 权力           | 内容                     | 效果               |
| -------------- | ------------------------ | ------------------ |
| **游戏节奏权** | 让玩家自主选择节奏       | 消除无聊，心流调节 |
| **难度选择权** | 让玩家选择挑战难度       | 失败时内部归因     |
| **身份定义权** | 自定义"我"在游戏中的投影 | 建立投影+禀赋效应  |
| **体验选择权** | 自主选择内容/顺序        | 满足不同口味       |

## 典型案例

| 游戏       | 下放的权力        | 设计                   |
| ---------- | ----------------- | ---------------------- |
| 王国保卫战 | 节奏权            | 提前召唤下一波敌人按钮 |
| 霰弹枪国王 | 难度选择权        | 优势+劣势组合选择      |
| 博德之门   | 身份定义权        | 自定义角色             |
| 浮游世界   | 体验选择权+难度权 | 不同难度分区自由前往   |

## 权力下放的归因修正

当玩家有选择权时，失败时更易内部归因（"我太贪心了"）而非外部归因（"数值不合理"），保护胜任感。

## 权力下放-节奏权（项目经验）

### 游戏节奏权让渡

让玩家自主控制战斗节奏，是权力下放中最直接、成本最低但效果最显著的形式：

| 形式          | 说明                    | 设计效果                       |
| :------------ | :---------------------- | :----------------------------- |
| **暂停/继续** | 时间相关一切暂停        | 玩家可在任何时刻思考策略       |
| **倍速切换**  | 1× / 2× / 4×            | 想慢慢思考就1×，想快速推进就4× |
| **备战暂停**  | 下次战斗前暂停          | 提前规划下一轮构筑             |
| **环境减速**  | 查看/拖动卡牌时自动降速 | 决策时不受时间压力             |

### 设计要点

- **避免"设计师强制节奏"与"玩家偏好"之间的冲突**：不同玩家有不同的最佳节奏，让玩家自主调节比设计单一节奏更合理
- **决策质量 > 反应速度**：卡牌构筑的核心乐趣是策略思考，不是手速比拼
- **倍速不改变游戏逻辑**：倍速只是加速播放，不影响任何概率/判定/冷却计算

### 项目实践

参考增量自走棋的操作控制系统（`项目经验/06-增量自走棋-经济与战斗分路设计.md`），暂停/倍速/备战暂停构成了完整的节奏控制体系，让玩家可以自主调节心流曲线。
