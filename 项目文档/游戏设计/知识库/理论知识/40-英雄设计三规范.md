# 电竞教父——英雄设计三规范

> 来源：千水《游戏创作心灵鸡汤》#2，以电竞教父项目经验提炼的英雄设计原则。

## 三种英雄设计规范

| 规范                   | 说明                                                         | 反例                            |
| :--------------------- | :----------------------------------------------------------- | :------------------------------ |
| **英雄本身不能全面**   | 必须有明确的强项和弱项，不能"什么都能做"                     | 初版剑魔：有复活、有AOE、有控制 |
| **英雄需要有明确特征** | 每个英雄要有独特的机制标签，让玩家一提到该特征就想到这个英雄 | 盲僧→操作上限、机器人→勾子      |
| **技能不能重复**       | 同一个英雄的技能组应该覆盖不同的使用场景，而非重复同一套逻辑 | 旧版EZ：Q和W都是"射一发的技能"  |

## 平衡性定义

> **平衡性** = 让游戏中的选择有价值，存在意义。

霍布森选择的反面：不是让所有选项相等，而是让每个选项在**特定场景**下都有被选择的理由。
