# 自然创作观

> 来源：千水《游戏创作心灵鸡汤》#19，结合乔纳森·布洛（Jonathan Blow）GDC 演讲《游戏设计的真相》及任天堂设计方法论。

## 一、核心理念

**设计不是发明，是发现。** 

> "文章本天成，妙手偶得之。"——好的设计不是我们创造的，而是我们在探索中发现的。

> "从某种角度来看，不是我去设计了游戏，而是游戏设计了它自己。"——乔纳森·布洛

> "设计不是一种技能，而是捕捉事物本质的感觉能力和洞察能力。"——原研哉

### 对游戏设计的意义

| 实践层面                       | 含义                                                           |
| :----------------------------- | :------------------------------------------------------------- |
| **追随乐趣（Follow the Fun）** | 制作原型 → 游玩 → 发现有趣的部分 → 扩展它                      |
| **原型先行**                   | 不要纸上谈兵，先做出来看它是否好玩                             |
| **接受失败**                   | 一个设计今天没有人提出来，明天也会有别人发现。创作者自我不重要 |
| **观察与训练**                 | 玩别人的游戏、解析它们，训练自己的感知能力                     |

---

## 二、与 Follow the Fun 的关系

任天堂的核心方法论——自然创作观的最佳实践：

| 阶段         | 做法                     | 案例                         |
| :----------- | :----------------------- | :--------------------------- |
| **提出想法** | 从核心玩法原型出发       | 马里奥 → 跳跃                |
| **制作原型** | 快速实现，看它是否有趣   | 瓦里奥制造→唱片机Demo        |
| **追随乐趣** | 哪里有趣扩展哪里         | 岩田聪"一边喊无聊一边玩下去" |
| **修剪**     | 去除不符合核心体验的元素 | 宫本茂→去掉了马里奥中的弹幕  |

> **自然创作观 vs 设计规范**：规范告诉你"怎么做"，自然创作观告诉你"怎么发现"。两者不矛盾——规范是工具，自然创作观是方向。

---

## 三、创作者的自我

在自然创作观下，**创作者的自我是不重要的**：

- 你是在发现乐趣，而不是发明乐趣
- 今天没有张三提出这个设计，李四明天也会探索到它
- 游戏设计本身有客观规律——**我们无法发明规律，只能发现规律**

### 这与"设计第一课"的关系

千水#13 中说：**"做游戏不是为了成功，是为了失败。"**

- 创作＝探索未知领域
- 如果你的作品和别人一致，你的存在对游戏门类毫无价值
- 失败是学习过程，而非结果的否定
