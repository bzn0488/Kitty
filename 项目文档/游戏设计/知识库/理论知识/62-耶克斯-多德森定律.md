# 耶克斯-多德森定律（Yerkes-Dodson Law）

> 标签：`flow`, `rhythm`, `player-psychology`
> 关联概念：`36-心流通道.md`, `43-游戏节奏设计.md`, `20-难度调节方案.md`
> 来源：Robert M. Yerkes & John D. Dodson（1908）
> 完整展开见：`游戏设计理论/04-游戏节奏与心流设计.md §1.1`

## 核心模型

唤醒水平（Arousal）与绩效表现之间存在倒U型曲线关系：

```
       表现
        ↑
   最佳  │    ●
        │   / \
        │  /   \
        │ /     \
        │/       \
        └────────────→ 唤醒水平
          低    高
```

- **唤醒水平过低** → 无聊、注意力涣散
- **唤醒水平适中** → 最佳表现（心流状态的核心生理基础）
- **唤醒水平过高** → 焦虑、紧张、认知功能下降

## 任务复杂度的影响

| 任务类型          | 最佳唤醒水平 | 设计含义                                |
| :---------------- | :----------- | :-------------------------------------- |
| **简单/熟练任务** | 较高         | 竞技/刷怪等操作需要更高唤醒才能进入状态 |
| **复杂/陌生任务** | 较低         | 策略思考/Build构筑需要低唤醒以保持专注  |

## 与心流通道的关系

耶克斯-多德森定律是心流理论的**生理学基座**：

- 心流通道中的「焦虑区」对应高唤醒低表现区
- 心流通道中的「无聊区」对应低唤醒区
- 两者的「通道」正是最佳唤醒水平区间

关键区别：心流理论强调**技能与挑战的匹配**，耶克斯-多德森强调**唤醒水平与任务复杂度的匹配**——两者互为因果。

## 皮质醇的调节作用

2007年Lupien等人的研究发现：
- 记忆表现 vs 糖皮质激素水平也呈倒U型曲线
- 轻度升高时长期增强作用（LTP）最佳
- 过高或过低都会显著降低LTP

触发应激反应的四种情境：
1. **新颖**（novel）
2. **不可预测**（unpredictable）
3. **不可控制**（uncontrollable by individual）
4. **社会评价威胁**（social evaluative threat）

> **设计启示**：在引入新机制/新挑战时，应避免同时触发多种应激因素——例如不要让玩家在**不可预测**且**不可控制**的环境中第一次面对**新颖**的Boss。

## 替代模型与局限

| 模型                             | 与耶克斯-多德森的关系                                            |
| :------------------------------- | :--------------------------------------------------------------- |
| **心流理论（Csikszentmihalyi）** | 更精细的"技能-挑战"匹配模型，是耶克斯-多德森在游戏设计中的进化版 |
| **逆转理论（Reversal Theory）**  | 认为心理运作基于双稳态而非稳态，与耶克斯-多德森的稳态假设不同    |

## 设计应用

| 场景          | 策略                   | 示例                                 |
| :------------ | :--------------------- | :----------------------------------- |
| **教程期**    | 保持低唤醒，简单任务   | 《黑魂》的初期小怪让玩家熟悉操作     |
| **Boss战**    | 提高唤醒，但不超过阈值 | Boss的出场动画/音效营造紧张感        |
| **策略决策**  | 降低唤醒（暂停/倍速）  | 自走棋的备战暂停降低决策时的唤醒水平 |
| **刷怪/跑图** | 维持中等唤醒           | 动态变化的敌人密度和奖励频率         |
