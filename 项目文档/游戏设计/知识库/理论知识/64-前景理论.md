# 前景理论（Prospect Theory）

> 标签：`player-psychology`, `cognitive-bias`, `decision-design`
> 关联概念：`46-正反馈与负反馈循环.md`, `28-体验4X.md`, `22-权力下放设计.md`
> 来源：Daniel Kahneman & Amos Tversky（1979），2002年诺贝尔经济学奖
> 完整展开见：`游戏设计理论/02-玩家心理学.md §14.1`

## 核心命题

前景理论是对"期望效用理论"的行为修正，描述人在**风险下的决策模式**。

### 参考点依赖（Reference Point Dependence）

人的决策不基于结果的**绝对值**，而基于与某个**参考点**的偏差。

```
同一笔 $500：
· 预期收益 $1000 只拿到 $500 → 感知为"损失"
· 预期收益 $0 拿到 $500     → 感知为"收益"
```

> **设计启示**：玩家的满意度取决于"预期 vs 实际"的对比，而非奖励的绝对数值。管理预期比提升数值更重要。

### 损失厌恶（Loss Aversion）

**损失的心理权重约为同等收益的 2~2.5 倍。**

```
损失 $100 的痛苦 ≈ 获得 $200~$250 的快乐
```

### 敏感性递减（Diminishing Sensitivity）

距离参考点越远，单位变化的影响越小：
- $100 → $200 的感知变化 > $1100 → $1200 的感知变化
- 这也是"边际递减"的心理根源之一

## 四重风险模式

|            | 收益域                   | 损失域                         |
| :--------- | :----------------------- | :----------------------------- |
| **高概率** | **风险规避**（宁可稳拿） | **风险寻求**（赌一把避免损失） |
| **低概率** | **风险寻求**（赌大奖）   | **风险规避**（宁可买个保险）   |

### 游戏设计中的应用

| 模式                  | 设计示例                                                |
| :-------------------- | :------------------------------------------------------ |
| 高概率收益 → 风险规避 | 玩家面对"90%概率获得奖励"时，倾向于**保底**而非赌博扩大 |
| 低概率收益 → 风险寻求 | 玩家面对"1%概率获得稀有物品"时，倾向于**赌一把**        |
| 高概率损失 → 风险寻求 | 即将失败时，玩家倾向"拼死一搏"（Boss残血时不放跑）      |
| 低概率损失 → 风险规避 | 玩家愿意为"1%概率掉装备"购买保险/复活道具               |

## 概率权重扭曲（Probability Weighting）

人**高估小概率事件**，**低估大概率事件**：

```
· 1% 的客观概率 → 心理感知约为 ~5%
· 99% 的客观概率 → 心理感知约为 ~95%
```

> **设计启示**：1% 掉率的稀有物品在玩家心中的期待值远超数学期望——这既可以用来创造"惊喜"（投篮效应），也可能导致对稀有度的过度追求。

## 前景理论 vs 过度理由效应

| 维度         | 前景理论          | 过度理由效应      |
| :----------- | :---------------- | :---------------- |
| **适用范围** | 风险决策时        | 行为动机归因时    |
| **核心机制** | 参考点 + 损失厌恶 | 外驱替代内驱      |
| **工具定位** | 审查已完成的系统  | 设计奖励/任务系统 |

## 审查要点（设计完成后自检）

| 问题                                           | 说明                                 |
| :--------------------------------------------- | :----------------------------------- |
| 奖励的参考点是否清晰？玩家预期是否被管理好了？ | 预期落差是负反馈的主要源头           |
| 是否有利用损失厌恶的设计？是否过度了？         | 适度利用可增加参与度，过度则引发反感 |
| 小概率事件是否被合理使用？                     | 低概率高回报是"投篮效应"的理论基础   |
| 惩罚设计的损失感知是否被充分考虑？             | 玩家对惩罚的敏感度是奖励的2倍        |
