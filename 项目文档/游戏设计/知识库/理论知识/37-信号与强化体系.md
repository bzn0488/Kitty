# 信号与强化体系

> 标签：`player-psychology`, `reinforcement`
> 关联概念：`09-过度理由效应.md`, `46-正反馈与负反馈循环.md`, `26-双轨收益与强化程式.md`

## 信号（巴甫洛夫条件反射）

将**信号**与**奖励**反复关联，最终信号本身引发期待。

### 设计应用
- 不直接展示奖励，将奖励包裹成信号
- ★ 外在驱动力→可被"包装"成内在驱动力
- 适合在潜在关联链路使用（打败Boss→奖励、探索→奖励）

## 斯金纳强化程式

| 程式         | 规则                         | 行为模式                     |
| ------------ | ---------------------------- | ---------------------------- |
| 连续强化     | 每次都奖励                   | 学习最快，消退最快           |
| 固定比率     | 固定次数后奖励               | 频率高，强化后短暂停顿       |
| **可变比率** | **平均次数后奖励（不固定）** | **频率最高、最稳定、最持久** |
| 固定时距     | 固定间隔后第一次行为奖励     | 临近时激增，刚获时骤降       |
| 可变时距     | 不固定间隔后第一次行为奖励   | 稳定中偏高，消退慢           |

### 鸽子实验启示
给予奖励时，必须**清晰告知玩家奖励为何而来**。否则玩家归因到错误行为 → 预期落空 → 负反馈。

## 强化与惩罚的四象限矩阵（补充）

> 完整展开见：`游戏设计理论/02-玩家心理学.md §7.1`

斯金纳操作性条件反射中，行为后果按照两个维度分为四种类型：
- **正向/负向**：刺激是**添加**还是**移除**
- **强化/惩罚**：行为是被**加强**还是**减弱**

```
                  添加刺激 (+)              移除刺激 (-)
              ┌───────────────────┬───────────────────┐
  强化行为    │   ✅ 正强化         │   🔄 负强化         │
  (行为增多)  │   给予愉悦刺激       │   移除厌恶刺激       │
              │   例: 击败Boss掉装备  │   例: 吃药解除中毒    │
              ├───────────────────┼───────────────────┤
  惩罚行为    │   ⚠️ 正惩罚         │   ❌ 负惩罚         │
  (行为减少)  │   施加厌恶刺激       │   移除愉悦刺激       │
              │   例: 踩陷阱扣血     │   例: 死亡掉落装备    │
              └───────────────────┴───────────────────┘
```

### 游戏设计中的特别注意

**一、慎用「负惩罚」（移除奖励）**
- 与"损失厌恶"原理一致——玩家对损失的敏感度远高于获得同等收益
- 移除已有奖励会触发强烈的负反馈
- 推荐使用**小的正惩罚**（如少量扣血）而非移除奖励

> 例外：Roguelike/硬核生存中死亡惩罚可作为核心体验，但这是玩家"知情同意"的范畴。

**二、注意玩家的心理预期**
- 反馈的方向应与玩家预期大致一致
- 幅度可以适当超出预期（惊喜），但不宜反向
- 这与鸽子实验的教训一致——奖励为何而来必须明确

### 塑造（Shaping）

塑造是指通过**逐步强化越来越接近目标的行为**，最终让个体学会复杂行为。

**在游戏设计中的应用**：渐进式教程——先教基础跳跃 → 正反馈 → 再教二段跳 → 正反馈 → 再教墙跳。每一步都给予及时的、符合预期的正反馈，与 SDT 中的「胜任感」直接相关。

## 强化模式的选择策略

| 目标                 | 推荐程式     |
| -------------------- | ------------ |
| 快速学习（教程期）   | 连续强化     |
| 维持高频行为         | 固定比率     |
| **长期维持（最佳）** | **可变比率** |
| 规律性参与           | 固定时距     |
