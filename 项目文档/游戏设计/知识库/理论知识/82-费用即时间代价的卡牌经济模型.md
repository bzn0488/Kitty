# 费用即时间代价的卡牌经济模型

> 来源：荒野与曙光 GDD 知识分析——时间轴驱动战斗系统的核心设计特征
> 关联概念：`16-卡牌设计规范.md`, `25-杀戮尖塔卡牌设计分析.md`, `心流通道.md`, `游戏节奏设计.md`
> 与现有知识库的关系：补充 `16-卡牌设计规范.md` 的费用模型分类——新增第四种费用模型

---

## 一、定义

在传统卡牌游戏中，卡牌费用是**能量/法力消耗**——玩家积累资源后消耗。在时间轴驱动游戏中，卡牌费用的本质是**时间代价**——打出高费牌意味着推进更多时间，让怪物更接近其动作触发点。

```
传统模型：费用 = 能量消耗 → 限制你的出牌数量
时间模型：费用 = 时间代价 → 决定你愿意跳过多少个回合
```

## 二、三种费用模型的对比

|     模型     | 代表游戏           | 核心约束              | 玩家的权衡             | 设计特征                    |
| :----------: | ------------------ | --------------------- | ---------------------- | --------------------------- |
| **能量消耗** | 杀戮尖塔、MTG      | 每回合固定能量上限    | "我这回合能打出哪些牌" | 资源积累→消耗的循环         |
| **冷却/CD**  | The Bazaar、自走棋 | 卡牌使用后需冷却      | "我什么时机打出这张牌" | 单卡独立冷却，不共享资源    |
| **时间代价** | 荒野与曙光         | 推进时间轴 = 敌方向前 | "我愿意为这效果等多久" | 0费=立即结算，N费=推进N回合 |

## 三、时间代价模型的核心特征

### 3.1 费用的双重语义

| 语义         | 含义                        | 玩家理解         |
| ------------ | --------------------------- | ---------------- |
| **显性语义** | N费 = N回合后生效           | 数值化的等待时间 |
| **隐性语义** | N费 = 怪物动作触发点接近N步 | 风险随费用增长   |

> 高费卡不仅是"强卡"——它是"高风险高回报"的决策。费用的本质是**风险定价**而非**能量定价**。

### 3.2 0费卡的特殊地位

在时间代价模型中，0费卡具有完全不同的意义：

|   模型   | 0费卡的含义                                                 |
| :------: | ----------------------------------------------------------- |
| 能量模型 | 免费白嫖，不消耗资源                                        |
| 时间模型 | **不推进回合=不付出时间代价**——在时间焦虑型战斗中价值被放大 |

> ⚠️ **设计风险**：0费卡过多可能导致时间焦虑被消除，破坏时间轴驱动的核心体验。解决方案包括：
> - 严格控制0费卡的数量和效果强度
> - 引入"每轮限X张0费卡"的规则
> - 或让0费卡有微小的推进（如0.5回合）

### 3.3 回合序号作为设计锚点

时间代价模型引入了一个独特的坐标轴：**回合序号**。

传统模型中，回合是抽象的时间概念。在时间代价模型中，每个回合都有明确的序号和对应的事件：

```
"第3回合：怪物使用蓄力"
"第5回合：怪物冲撞"
"第7回合：弱点暴露"
```

这为卡牌设计提供了精确的触发条件坐标——"若已有精灵在场上，你的自然法术费用减少1点"这种条件不再成立，取而代之的是"第4回合后，你的卡牌伤害+2"。

## 四、设计要点

### 4.1 费用曲线的特殊设计

时间代价模型的费用曲线不同于能量模型：

| 费用区间 | 设计要点     | 说明                     |
| :------: | ------------ | ------------------------ |
|   0费    | 严格限制数量 | 白嫖打破时间焦虑，需控制 |
|  1-3费   | 常规战斗     | 核心战术区间             |
|  4-6费   | 高风险高回报 | 推进多回合，需足够强大   |
|   7费+   | 终结技       | 极慢，必须配得上风险     |

### 4.2 时间焦虑作为驱动力

与传统模型不同，时间代价模型的核心驱动力是**时间焦虑**：

- 玩家打出高费牌时，明知"怪物会靠近动作点"
- 这种焦虑本身就是策略乐趣的一部分
- 玩家在"收益"和"风险"之间做取舍

### 4.3 外部归因保护自我效能

当一轮包含的AV数由怪物决定（而非玩家自选）时：

- 短轮 → "不是我的牌不好，是这回合太短了"
- 长轮 → "运气好，有充足时间布局"
- 外部归因保护玩家的自我效能感

## 五、适用判断

| 适合使用时间代价模型               | 不适合使用时间代价模型 |
| ---------------------------------- | ---------------------- |
| 战斗有明确的时间压力（Boss计时器） | 纯策略/解谜型卡牌游戏  |
| 怪物行为有可预判的意图             | 快节奏竞技卡牌         |
| 攻防转换是核心乐趣                 | 单纯比大小的数值对抗   |
| 部位/子系统是战斗的焦点            | 纯构筑型DBG            |
