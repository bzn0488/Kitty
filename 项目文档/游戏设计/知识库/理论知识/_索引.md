## 关联关系图

```
设计哲学 ──→ 叙事设计 ──→ 宏观/微观层叙事
    │              │
    └──→ 权力下放 ──┘
                      │
玩家心理 ──→ 正负反馈 ──→ 快乐总量 ──→ 正反馈技法（协同/叠层/等）
    │              │
    └──→ 强化体系 ──┘
    ↓
心理力学模型 ←── 驱动力模型
    ↓
认知评价理论CET ←── SDT

心流通道 ──→ 游戏节奏 ──→ 难度调节
    │              │
    └──→ 权力下放 ──┘

涌现式 ──→ 系统交互 ──→ 玩具系统
    │              │
    └──→ 渐进式生成 ──→ 体验4X

Roguelite ──→ 局外成长 ──→ 规则变异
    │                        │
    └──→ 内容解锁 ──────────┘（替代方案）

CRPG ──→ 三层角色架构 ──→ 职业/技能系统
    │                  │
    └──→ 叙事交互系统 ──┘

相对最优解 ──→ 三层变化模型 ──→ L1问题变化
    │                      ├──→ L2解答方式变化
    │                      └──→ L3规则变化（最强）
    │
    └──→ 霍布森选择困境 ──→ 绝对最优解 vs 相对最优解

任务列表 ──→ Checklist旧范式 ──→ 被超越 ──→ 网状系统游戏
    │              │                              │
    ├──→ 分段奖励 ──┘               ┌──→ 种子机制 ──→ 鸟笼效应
    │                                │
    ├──→ 去目标化指引 ──→ 引力式设计 │──→ 物品即目标 ──→ 多层揭示
    │                                │
    ├──→ 驱动力模型 ──→ 外驱 vs 内驱 │──→ 系统交互 ──→ 破解蚂蚁区隔
    │                                │
    └──→ 强制循环 ──→ 强化程式       └──→ 世界状态条 ──→ 隐性进度
```
