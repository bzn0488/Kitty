# 玩家心理学与行为模型

> 整理自：`_原始文档归档/玩家行为分析.txt` + `心流.txt`（部分）

## 一、行为分析基础模型

玩家进行行为的本质：**追求正反馈（快乐）**，改变行为基于**减少不悦（负反馈）**。

**行为切换条件**：
```
新行为总驱动力 > 当前行为总驱动力
```

行为结果带来的内外驱动力 > 行为本身产生的内外驱动力

## 二、自我决定理论（SDT）

### 三大基本心理需求

| 需求 | 定义 | 设计要点 |
|------|------|----------|
| **胜任感** | 玩家感受到自己有能力克服挑战 | 数值强度+技巧强度，技巧强度验证带来更大正反馈 |
| **自主感** | 行为来源于自我意愿 | 权力下放、选择权让渡 |
| **归属感** | 与NPC或其他玩家建立情感纽带 | 情感联结、宏大叙事参与感 |

### 胜任感的两类

1. **数值强度**：升级、装备等显性成长
2. **技巧强度**：对机制理解、反应速度提升

★ 技巧强度的提升和验证能给玩家**更大正反馈**。可通过设计让玩家产生"技巧提升"的错觉（例：魂系列通过篝火+捷径+背板实现）。

### 自主感的实现：权力下放

| 权力类型 | 内容 | 效果 |
|----------|------|------|
| 游戏节奏权 | 玩家自主选择节奏（如提前召唤敌人） | 消除无聊，进入心流 |
| 难度选择权 | 让玩家选择难点 | 失败时归因自己而非系统 |
| 身份定义权 | 定义"我"指代的角色 | 建立投影，禀赋效应 |
| 体验选择权 | 自主选择内容和顺序 | 满足不同玩家口味 |

## 三、过度理由效应与水桶理论

### 3.1 过度理由效应
当玩家为某一行为找到强有力的外部理由时，会倾向于忽视或低估内在动机。

**设计启示**：希望内驱旺盛时，需要降低外驱强度。但外驱过低可能不足以驱使行动。

### 3.2 水桶理论（静态快照模型）

```
水桶 = 停止当前行为所需的阻力
先倒"外驱"水 → 再倒"内驱"水 → 水桶满了则切换行为
溢出的部分（内驱）流失
```

注：模型为静态快照，现实中玩家是动态的（疲劳、时间压力等影响水桶大小）。

## 四、信号与强化

### 4.1 信号
巴甫洛夫条件反射：将信号与奖励关联，让玩家产生期待感。
- ★ 将外在驱动力**包裹成**内在驱动力
- 适合用在潜在关联链路（打败Boss→奖励，探索→奖励）

### 4.2 斯金纳强化程式

| 强化程式 | 规则 | 行为模式 |
|----------|------|----------|
| 连续强化 | 每次都奖励 | 学习最快，消退也最快 |
| 固定比率 | 固定次数后奖励 | 频率高，强化后有短暂停顿 |
| **可变比率** | **平均次数后奖励（不固定）** | **频率最高、最稳定、最持久** |
| 固定时距 | 固定间隔后第一次行为奖励 | 临近时点激增，刚获奖励骤降 |
| 可变时距 | 不固定间隔后第一次行为奖励 | 稳定中偏高，消退慢 |

### 4.3 斯金纳鸽子实验启示
给予奖励时，必须**清晰告知玩家奖励为何而来**。否则玩家可能归因到错误行为，最终因预期落空产生负反馈。

## 五、动机的消退与保护

### 5.1 边际递减
同一种奖励反复出现且形式单一时，玩家获得的正反馈会显著减少。需要多样化奖励种类和频率。

### 5.2 习得性无助
多次尝试后仍然失败（巨大负反馈），导致"无论做什么都没有意义"的心理状态，胜任感消亡。

### 5.3 蔡格尼克效应
未完成的任务在心理上产生张力，使信息更容易被记忆。
- **未完成**：张力持续，信息易被提取
- **完成**：张力释放，记忆淡化

### 5.4 海明威效应
"在写得顺手的时候停下来"——**积极离场**。
- 不要让玩家因疲劳被迫下线
- 积极离场减少回归的心理阻力，极大增加回归驱动力

## 六、玩具系统

定义：有一定学习成本的、提供一定游戏深度的、**对推进进程非必要**的玩法体验模块。

| 要素 | 说明 |
|------|------|
| 非必要 | 剥离外驱，纯粹由内驱驱动 |
| 学习成本+深度 | 玩家在"学习→验证→技巧提升"循环中获得正反馈 |
| 奖励 | 提升效率而非必须 |

**如何驱动玩家开始玩玩具**：在接触前展示该系统带来的奖励。

例：《异星工厂》火车运输系统——不用火车也能通关，但火车提供长距离大量运力。
