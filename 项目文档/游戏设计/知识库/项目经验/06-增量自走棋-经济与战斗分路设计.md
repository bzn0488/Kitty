# 增量自走棋 — 经济与战斗分路设计

> 项目经验
> 来源：增量自走棋
> 关联知识：`理论知识/16-卡牌设计规范.md`, `理论知识/18-快乐总量.md`, `理论知识/19-浪费规避设计.md`

## 设计背景

增量自走棋作为"快乐总量 ÷ 进程"分母最小化的探索项目，核心挑战在于：如何让玩家的每一秒都花在有意义的决策上？经济与战斗分路的提出，就是为了解决这一问题——将"经济获取"这一通常独立于战斗体系的维度，嵌入到战场的空间决策中。

## 核心设计

### 经济单位规则

| 规则             | 说明                                            |
| :--------------- | :---------------------------------------------- |
| **无战斗属性**   | 经济单位无生命值、攻击力、防御力等战斗属性      |
| **不可被攻击**   | 敌方目标选择自动跳过经济单位                    |
| **无可替换技能** | 仅拥有 1 个固有技能产出金币                     |
| **占用战场格**   | 与战斗单位共用战场，占据格子尺寸                |
| **效果限制**     | 不得产出任何战斗数值（护盾/治疗/伤害/属性增益） |

### 放置上限

| 难度            | 经济单位上限 |
| :-------------- | :----------: |
| 初始            |     1 个     |
| 每通过 1 个难度 |    +1 个     |
| 最大上限        |     4 个     |

### 经济单位升星（内置任务）

经济单位的升星方式**不同于**战斗单位的合成升星：

- 每个经济单位自带一个累计型升星任务（如"累计产出 50 金币"）
- 任务进度在战斗中实时累积，进度条始终可见
- 任务完成后手动点击升级 → 星级+1 → 自动生成下一级任务
- 升星效果：金币产出量提升、冷却缩短、触发条件放宽

**设计意图**：使用经济单位的行为本身驱动升星 → 形成正反馈闭环。进度条持续可见 → 目标梯度效应。

## 设计经验总结

### 空间权衡（核心设计点）

经济单位的本质是让玩家在"战斗力"和"经济力"之间做有意义的取舍：

```
战场格（稀缺资源）
    ├── 战斗单位 → 提升即时战力 → 当前关卡更容易
    └── 经济单位 → 提升金币收入 → 后续回合更强
```

玩家需要判断：当前战力是否撑得住？如果答案是"撑得住"，则值得投入经济单位。这个决策本身创造了持续的乐趣密度。

### 安全垫效应

即使战斗不顺，经济单位持续提供基础金币流入，避免"打不过→没资源→更打不过"的负反馈螺旋。这是`19-浪费规避设计.md`中"失败但不白费"原则的具体实践。

### 尺寸与效率的关系

| 尺寸 | 空间代价 |  金币产出效率  | 升星任务门槛 |
| :--- | :------: | :------------: | :----------: |
| 1 格 |    低    |   低（稳定）   |      低      |
| 2 格 |    中    | 中（较高期望） |      中      |
| 多格 |    高    | 高（条件触发） |      高      |

尺寸越大 → 占据格子越多 → 产出效率越高 → 升星条件也更高。玩家需要判断"值不值得用这么大的格子放经济单位"。

### 获取渠道隔离

- 经济卡包与战斗卡包为独立页签、独立抽卡等级
- 玩家在获取经济卡时已经知道它是经济定位，不期望有即战力
- 经济卡不受"诅咒卡检验"约束

### 与其他自走棋经济系统的对比

增量自走棋的经济系统与经典自走棋（TFT/酒馆战棋）有本质差异：

| 维度         | 经典自走棋               | 增量自走棋                       |
| :----------- | :----------------------- | :------------------------------- |
| **经济来源** | 利息+连胜/连败+基础收入  | 经济单位的空间产出               |
| **经济消耗** | 买单位+刷新+升级人口     | 抽卡包+升星                      |
| **经济决策** | "存钱 vs 花钱"的时间权衡 | "经济单位 vs 战斗单位"的空间权衡 |
| **失败补偿** | 连败奖励+选秀优先        | 经济单位持续产出（安全垫）       |
| **经济上限** | 50金利息上限             | 经济单位放置上限                 |

**设计启示**：增量自走棋的"空间权衡"模型是经典自走棋"时间权衡"模型的补充——两者可以结合使用，让经济系统拥有更丰富的决策维度。
