# 增量自走棋 & 荒野与曙光 — 卡牌设计检核

> 项目经验
> 来源：增量自走棋、荒野与曙光

## 卡牌系统设计原则

- 卡牌效果应有明确的成本收益模型
- 卡牌稀有度与效果强度正相关
- 每张卡牌应有独特定位，避免功能重叠

## 自闭环检测标准

**定义**：自闭环指一张卡移除场上所有其他队友后，仍能满额运作自身所有效果。

**检测方法**：把场上其他所有单位移除后，该单位是否仍能满额运作？
- 是 → ❌ 自闭环（删除）
- 否 → ✅ 需要队友参与，通过

**常见自闭环类型**：
| 类型          | 示例                                         |          判定           |
| ------------- | -------------------------------------------- | :---------------------: |
| 纯自我增益    | "自己+1攻击力"                               |        ❌ 自闭环         |
| 自我复制/增殖 | 自己生成自己的复制体                         |        ❌ 自闭环         |
| 依赖自身属性  | "以自身最大生命值的30%造成伤害"              |        ❌ 自闭环         |
| 依赖外部状态  | "相邻每有一个友方，攻击力+1"                 |         ✅ 通过          |
| 一次性结算型  | "上场时为同列友方加生命值"（结算后不再持续） | ✅ 通过（结算型≠自闭环） |

**例外**：经济单位不受此规则约束——经济单位的设计目的就是独立于战斗体系运作。

## 低稀有度单位底线原则

### 诅咒卡检验
设计完成后自问："如果三选一里只有这一张，我会选它吗？"

### 兜底攻击模式
1级（普通）单位必须有即时生效的行动（每N秒伤害/治疗/护盾）。纯条件触发/纯被动不能作为1级单位的唯一效果。

**可学习替代模式**：当1级单位的主导技能因条件不足无法生效时，提供一个基础弱主动作为兜底。条件满足后基础主动被覆盖/替换为更强效果。
- 示例：小兵→基础每6秒1点伤害→有高星同名时替换为"学习"复制技能
- 兜底攻击必须满足诅咒卡检验，即使很弱也不能让玩家空过回合

### 例外
- 2级及以上可以适当放宽纯条件触发
- 经济卡不受诅咒卡检验约束（通过独立渠道获取，玩家已知其经济定位）

## 充能 vs 冷却缩减 术语区分

| 概念         | 定义                                | 示例                                    |
| ------------ | ----------------------------------- | --------------------------------------- |
| **充能**     | 使目标技能的**当前冷却计时**前进N秒 | "为所有玩具兵团充能1秒"→CD计时器推进1秒 |
| **冷却缩减** | 减少技能**本身的CD属性值**          | "冷却时间减少10%"→技能CD从10秒变为9秒   |

- 充能不改变技能CD属性，只加速计时
- 冷却缩减永久改变技能CD属性
- 两者在Build中产生不同信号：充能=加速现有节奏，冷却缩减=提升长期效率

## 经济卡与战斗卡分路设计原则

### 获取渠道隔离
- 经济卡与战斗卡通过不同的渠道获取（专属卡包、独立商店格、事件奖励等）
- 玩家在获取经济卡时已知其经济定位

### 设计检验
- 经济效果是否与场上单位数量/Tag联动，而非固定值？
- 经济卡是否定位清晰（要么经济，要么战斗，不兼得）？
- 玩家是否会为了前期的经济牺牲战力去拿它？

## 数值平衡通用规则

- 同类卡牌DPS差距 ≤ 10%
- 消耗品价值按稀有度递增
- 卡牌组合协同不宜过度依赖单卡
