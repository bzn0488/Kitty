## 分类

| 类别           | 路径            | 说明                                                      |
| -------------- | --------------- | --------------------------------------------------------- |
| 📖 游戏设计理论 | `游戏设计理论/` | 综合大文档（教材式），从 `00-总览与阅读导引.md` 开始      |
| 📖 理论知识     | `理论知识/`     | 游戏设计基础理论，每概念独立文档 → 读 `理论知识/_索引.md` |
| 🧪 项目经验     | `项目经验/`     | 单项目提炼的经验，待多项目验证 → 读 `项目经验/_索引.md`   |
| 🔗 跨项目知识   | `跨项目知识/`   | 多项目验证后的通用设计规则 → 读 `跨项目知识/_索引.md`     |

## 晋升路径

```
项目经验/ ← 单项目原始提炼
    ↓ 多项目比较+归纳
跨项目知识/ ← 提炼后的通用规则
    ↓ 长时间验证
游戏设计理论/ + 理论知识/ ← 验证无误的设计原理
```
