# UI设计理论 - 索引

> 📌 游戏 UI/UX 设计相关的理论、原则和方法论。
> 按标签路由，配合 `理论知识/` 的交叉引用。

## 目录结构

```
UI设计理论/
├── _索引.md                ← 本文件
├── 01-游戏UI设计原则.md      ← 基础原则（Fitts/Hick/一致性/反馈/可见性）
├── 02-HUD信息架构.md         ← 信息优先级、视觉层次、屏幕空间
├── 03-交互反馈与动效设计.md   ← 反馈链路、动效引导、操作确认
├── 04-游戏UX评估方法.md      ← 启发式评估、可用性测试、认知走查
├── 05-状态呈现设计.md        ← 玩家状态可视化、态势感知
├── 06-菜单与元UI设计.md      ← 背包/商店/技能树、信息检索
├── 07-DiegeticUI与叙事界面.md← 游戏内界面、叙事融合
├── 08-女神异闻录5UI设计分析.md← 案例（风格化UI/叙事UI/菜单动画）
├── 09-旷野之息UI设计分析.md   ← 案例（隐形UI/世界即UI/极简HUD）
├── 10-HadesUI设计分析.md      ← 案例（战斗UI/恩赐三选一/渐进披露）
└── 11-操作映射与手感设计.md   ← 按键映射、控制器布局、响应延迟
```
